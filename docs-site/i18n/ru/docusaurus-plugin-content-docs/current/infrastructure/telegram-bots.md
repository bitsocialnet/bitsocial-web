# Телеграм-боты

Боты Bitsocial Telegram отслеживают списки клиентских сообществ в сети Bitsocial и автоматически пересылают новые сообщения в каналы Telegram. Каждое пересылаемое сообщение включает встроенные кнопки, которые ссылаются на исходное сообщение на 5chan и Seedit.

- **GitHub**: [bitsocialnet/bitsocial-telegram-боты](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## Доступные боты

| Бот             | Статус      | Описание                                                          |
| --------------- | ----------- | ----------------------------------------------------------------- |
| **Лента 5chan** | Активный    | Мониторит все каталоги 5chan и пересылает новые посты в Telegram. |
| **Фид Seedit**  | Планируется | Обеспечит ту же функциональность для сообществ Seedit.            |

## Настраивать

### Предварительные условия

- Node.js
- Пряжа
- Токен бота Telegram (создайте его через [БотОтец](https://t.me/BotFather))

### Установка

Клонируйте репозиторий и установите зависимости:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### Конфигурация

Создайте в корне проекта файл `.env` с токеном вашего бота:

```env
BOT_TOKEN=your_telegram_bot_token
```

### Бег

Запустите бота после настройки среды:

```bash
yarn start
```

## Формат сообщения

Когда бот пересылает сообщение в Telegram, он включает в себя две встроенные кнопки:

- **Просмотр на 5chan** – открывает публикацию в веб-клиенте 5chan.
- **Просмотр в Seedit** – открывает публикацию в веб-клиенте Seedit.

Это позволяет подписчикам Telegram сразу перейти к полной ветке обсуждения любого клиента, который они предпочитают.
