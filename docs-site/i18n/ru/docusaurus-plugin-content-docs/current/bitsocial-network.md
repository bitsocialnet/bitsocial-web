# Битсоциальная сеть

Bitsocial Network — это имя, используемое в генеральном плане для общего уровня цепочки приложений, который будет находиться под приложениями Bitsocial.

Идея не в том, чтобы заменить одноранговый социальный слой. Целью является предоставление общих экономических и координационных примитивов, которые понадобятся социальным приложениям, если они собираются конкурировать с существующими платформами в масштабе.

## Что значит власть

- децентрализованные домены Bitsocial, такие как `.bso`
- общая ликвидность между приложениями
- прочные рельсы монетизации
- финансовые структуры, которые банкам или платформам труднее задушить
- сетевые эффекты, которые не зависят от того, что одна компания владеет всем стеком

## Почему это важно

Децентрализация сообществ и идентичностей необходима, но недостаточно децентрализовать все социальные сети.

Если экономический уровень по-прежнему зависит от нескольких централизованных рельсов, то приложения по-прежнему легко подвергаются давлению, деплатформации или финансовому голоданию. Bitsocial Network — это предлагаемый ответ на этот второй уровень зависимости.

## Отношения с остальной частью Bitsocial

Bitsocial Network предлагается как общая инфраструктура для приложений Bitsocial, а не как продукт, с которым пользователи взаимодействуют напрямую каждый день.

Публичным результатом будет:

- сообщества остаются одноранговыми
- приложения остаются дифференцированными
- монетизация и нейминг становятся более долговечными
- ценность может перемещаться по экосистеме без воссоздания централизованного владельца платформы.

## Почему это находится на этапе 4?

В генеральном плане это размещается после предыдущих доказательств на уровне приложения:

1. децентрализовать имиджборды
2. запустить Bitsocial RPC
3. децентрализовать форумы
4. запустить битсоциальную сеть

Эта последовательность имеет смысл, поскольку уровень координации на уровне экосистемы становится более полезным, когда в сети уже существуют реальные приложения и реальное поведение пользователей.
