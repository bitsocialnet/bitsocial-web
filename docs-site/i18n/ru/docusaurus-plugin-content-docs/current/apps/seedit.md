# сеять

Seedit — это бессерверная, не требующая администратора и полностью децентрализованная альтернатива Reddit старого стиля, построенная на протоколе Bitsocial. Это дает пользователям полную власть над своими сообществами — любой может создать одно, и несколько сообществ могут конкурировать за места в листинге по умолчанию.

## Как работают сообщества

Seedit организует контент в сообщества, похожие на субреддиты. Каждое сообщество принадлежит своему создателю и функционирует без централизованного управления. Когда несколько сообществ нацелены на один и тот же слот по умолчанию, они могут конкурировать за эту позицию.

### Создание сообщества

Создать новое сообщество можно двумя способами:

1. **Клиент для настольных компьютеров**. Используйте встроенный графический интерфейс настольного приложения для настройки вашего сообщества.
2. **bitsocial-cli** — запустите инструмент CLI для создания сообществ и управления ими из командной строки.

### Список сообществ по умолчанию

Список сообществ по умолчанию, отображаемый на главной странице, определен в `default-multisub.json` в репозитории списков. Чтобы добавить ваше сообщество к настройкам по умолчанию, отправьте запрос на включение с деталями вашего сообщества.
