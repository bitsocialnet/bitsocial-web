# روبوتات برقية

تقوم روبوتات Bitsocial Telegram بمراقبة قوائم مجتمع العملاء على شبكة Bitsocial وإعادة توجيه المنشورات الجديدة تلقائيًا إلى قنوات Telegram. تتضمن كل رسالة مُعاد توجيهها أزرارًا مضمنة تربطك بالمنشور الأصلي على 5chan وSeedit.

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## الروبوتات المتاحة

| بوت                | الحالة | الوصف                                                             |
| ------------------ | ------ | ----------------------------------------------------------------- |
| ** تغذية 5chan **  | نشط    | يراقب جميع أدلة 5chan ويعيد توجيه المنشورات الجديدة إلى Telegram. |
| ** تغذية Seedit ** | مخطط   | سيوفر نفس الوظيفة لمجتمعات Seedit.                                |

## الإعداد

### المتطلبات الأساسية

- Node.js
- غزل
- رمز بوت تيليجرام (قم بإنشاء واحد عبر [بوتالأب](https://t.me/BotFather))

### التثبيت

استنساخ المستودع وتثبيت التبعيات:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### التكوين

قم بإنشاء ملف `.env` في جذر المشروع باستخدام رمز الروبوت الخاص بك:

```env
BOT_TOKEN=your_telegram_bot_token
```

### الجري

ابدأ تشغيل الروبوت بعد تكوين بيئتك:

```bash
yarn start
```

## تنسيق المشاركة

عندما يقوم الروبوت بإعادة توجيه منشور إلى Telegram، فإنه يتضمن زرين مضمنين:

- **عرض على 5chan** - يفتح المنشور في عميل الويب 5chan.
- **عرض على Seedit** -- يفتح المنشور في عميل الويب Seedit.

يتيح ذلك لمشتركي Telegram الانتقال مباشرة إلى سلسلة المناقشة الكاملة على أي عميل يفضلونه.
