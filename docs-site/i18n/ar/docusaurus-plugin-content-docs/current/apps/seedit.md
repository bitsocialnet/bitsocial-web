# Seedit

Seedit هو بديل بدون خادم، وبدون إدارة، ولا مركزي تمامًا لـ Reddit ذي الطراز القديم، والمبني على بروتوكول Bitsocial. إنه يمنح المستخدمين ملكية كاملة لمجتمعاتهم - يمكن لأي شخص إنشاء مجتمع، ويمكن لمجتمعات متعددة التنافس على فتحات القائمة الافتراضية.

## التنزيلات

| منصة        | الرابط                                         |
| ----------- | ---------------------------------------------- |
| ويب         | [Seedit.app](https://seedit.app)               |

| سطح المكتب  | متوفر لأنظمة التشغيل Mac وWindows وLinux       |
| الجوال      | متاح للأندرويد                                 |

## كيف تعمل المجتمعات

ينظم Seedit المحتوى في مجتمعات مشابهة لـ subreddits. كل مجتمع مملوك لمنشئه ويعمل بدون إدارة مركزية. عندما تستهدف مجتمعات متعددة نفس الفتحة الافتراضية، يمكنها التنافس على هذا المركز.

### إنشاء مجتمع

هناك طريقتان لإنشاء مجتمع جديد:

1. **عميل سطح المكتب** — استخدم واجهة المستخدم الرسومية المضمنة في تطبيق سطح المكتب لإعداد مجتمعك وتكوينه.
2. **bitsocial-cli** — قم بتشغيل أداة واجهة سطر الأوامر (CLI) لإنشاء المجتمعات وإدارتها من سطر الأوامر.

### قائمة المجتمع الافتراضية

يتم تعريف قائمة المجتمعات الافتراضية التي تظهر على الصفحة الأولى في `default-multisub.json` ضمن مستودع القوائم. لإضافة مجتمعك إلى الإعدادات الافتراضية، أرسل طلب سحب يتضمن تفاصيل مجتمعك.
