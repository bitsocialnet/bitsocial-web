# i18n 기본 언어 정책

익명 방문자는 이제 다음 순서로 언어를 확인합니다.

1. `?lang=` 쿼리 매개변수
2. `localStorage`에서 저장된 언어 선택기 선택
3. 지역 인식 익명 기본값:
   - 명시적으로 나열된 영어에 영어 강제 적용-heavy 지역
   - 그렇지 않으면 지원되는 로케일 중 하나인 경우 브라우저/장치 언어를 사용
4. `en`로 대체

## 소스 기준

- [EF 영어 능력 지수 2025](https://www.ef.edu/epi/) for 국가 수준 영어 능력
- [2020년 기록 인구조사](싱가포르의 https://www.singstat.gov.sg/modules/infographics/census-2020), 5세 이상 거주자의 48.3%가 가장 자주 사용하는 모국어는 영어였습니다.
