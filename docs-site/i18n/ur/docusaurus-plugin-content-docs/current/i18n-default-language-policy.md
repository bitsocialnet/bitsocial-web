# i18n ڈیفالٹ لینگویج پالیسی

گمنام زائرین اب اس ترتیب میں زبان کو حل کرتے ہیں:

1. `?lang=` استفسار پیرام
2. `localStorage` سے محفوظ کردہ زبان کے انتخاب کنندہ کا انتخاب
3. علاقہ سے آگاہ گمنام ڈیفالٹ:
   - واضح طور پر درج انگریزی والے خطوں میں انگریزی پر زور دیں۔
   - بصورت دیگر براؤزر/آلہ کی زبان استعمال کریں جب یہ ہمارے تعاون یافتہ لوکلز میں سے ایک ہو۔
4. `en` پر واپس جائیں۔

## ماخذ بیس لائن

- [ای ایف انگلش پرافینسی انڈیکس 2025](https://www.ef.edu/epi/) ملکی سطح کی انگریزی کی مہارت کے لیے
- [سنگاپور کی مردم شماری 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) سنگاپور کے لیے، جہاں 5+ سال کی عمر کے 48.3% رہائشیوں کے لیے انگریزی سب سے زیادہ بولی جانے والی گھریلو زبان تھی۔
