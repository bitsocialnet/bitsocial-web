# seedit

سیڈٹ پرانے طرز کے Reddit کا ایک سرور لیس، ایڈمن لیس، اور مکمل طور پر وکندریقرت متبادل ہے، جو بٹسوشل پروٹوکول پر بنایا گیا ہے۔ یہ صارفین کو ان کی کمیونٹیز پر مکمل ملکیت فراہم کرتا ہے — کوئی بھی ایک بنا سکتا ہے، اور متعدد کمیونٹیز ڈیفالٹ لسٹنگ سلاٹس کے لیے مقابلہ کر سکتی ہیں۔

## ڈاؤن لوڈز

| پلیٹ فارم | لنک                                              |
| --------- | ------------------------------------------------ |
| ویب       | [seedit.app](https://seedit.app)                 |
| Web (ENS) | `seedit.eth` بذریعہ بہادر براؤزر یا IPFS کمپینین |
| ڈیسک ٹاپ  | میک، ونڈوز اور لینکس کے لیے دستیاب               |
| موبائل    | اینڈرائیڈ کے لیے دستیاب ہے۔                      |

## کمیونٹیز کیسے کام کرتی ہیں۔

سیڈٹ مواد کو سبریڈیٹس کی طرح کمیونٹیز میں ترتیب دیتا ہے۔ ہر کمیونٹی اس کے خالق کی ملکیت ہے اور مرکزی انتظامیہ کے بغیر کام کرتی ہے۔ جب متعدد کمیونٹیز ایک ہی ڈیفالٹ سلاٹ کو نشانہ بناتے ہیں، تو وہ اس پوزیشن کے لیے مقابلہ کر سکتے ہیں۔

### کمیونٹی بنانا

ایک نئی کمیونٹی بنانے کے دو طریقے ہیں:

1. **ڈیسک ٹاپ کلائنٹ** — اپنی کمیونٹی کو سیٹ اپ اور کنفیگر کرنے کے لیے ڈیسک ٹاپ ایپلیکیشن میں بلٹ ان GUI استعمال کریں۔
2. **bitsocial-cli** — کمانڈ لائن سے کمیونٹیز بنانے اور ان کا نظم کرنے کے لیے CLI ٹول چلائیں۔

### ڈیفالٹ کمیونٹی لسٹ

کمیونٹیز کی فہرست پہلے سے طے شدہ صفحہ پر ظاہر ہوتی ہے۔ `default-multisub.json` فہرستوں کے ذخیرے کے اندر۔ اپنی کمیونٹی کو ڈیفالٹس میں شامل کرنے کے لیے، اپنی کمیونٹی کی تفصیلات کے ساتھ ایک پل کی درخواست جمع کروائیں۔
