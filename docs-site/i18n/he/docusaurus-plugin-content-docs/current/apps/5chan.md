# 5chan

5chan הוא לוח תמונות ללא שרתים, ללא ניהול ומבוזרי לחלוטין הפועל על פרוטוקול Bitsocial. הוא עוקב אחר מבנה ספריות ה-imageboard המוכר תוך הצגת בעלות מבוזרת - כל אחד יכול ליצור לוח, ולוחות מרובים יכולים להתחרות על אותה משבצת ספרייה באמצעות מנגנון הצבעה.

## הורדות

| פלטפורמה    | קישור                          |
| ----------- | ------------------------------ |
| אינטרנט     | [5chan.app](https://5chan.app) |
| שולחן עבודה | זמין עבור Mac, Windows ולינוקס |
| נייד        | זמין עבור אנדרואיד             |

## איך לוחות עובדים

5chan מארגן תוכן בלוחות באמצעות פריסת ספרייה קלאסית (למשל, `/b/`, `/g/`). בניגוד ללוחות תדמית מסורתיים שבהם מנהל מרכזי שולט בכל לוח, 5chan מאפשר לכל משתמש ליצור ולבעלות מלאה את הלוח שלו. כאשר לוחות מרובים מכוונים לאותו משבצת ספרייה, הם מתחרים על תפקיד זה באמצעות הצבעה.

### יצירת לוח

כדי ליצור לוח חדש, עליך להפעיל את `bitsocial-cli` כצומת עמית לעמית. זה מבטיח שהלוח שלך מתארח בצורה מבוזרת מבלי להסתמך על שרת מרכזי כלשהו.

### מטלות ספריות

הקצאות חריצי ספריות (איזה לוח מופיע באיזה נתיב) מנוהלות כעת באמצעות בקשות משיכה של GitHub לקובץ `5chan-directories.json`. זהו תהליך זמני - מהדורות עתידיות יתמכו ביצירת לוח בתוך האפליקציה ובהצבעה מבוססת-פאבים כדי לטפל בהקצאות ספריות באופן אוטומטי.

## פנימיות

מתחת למכסה המנוע, 5chan משתמש בשכבת ה-API של plebbit-js עבור אינטראקציות הפרוטוקול שלה. כפי שצוין באזהרה לעיל, הפניות פנימיות אלה עדיין נושאות שמות מדור קודם שקודם למותג מחדש של Bitsocial.
