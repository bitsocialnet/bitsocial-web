# i18n מדיניות ברירת מחדל לשפות

מבקרים אנונימיים פותרים כעת את השפה בסדר הזה:

1. פרמטר שאילתה `?lang=`
2. בחירת בורר שפה נשמרה מ-`localStorage`
3. ברירת מחדל אנונימית המודעה לאזור:
   - לכפות אנגלית באזורים כבדים באנגלית ברשימה מפורשת
   - אחרת השתמש בשפת הדפדפן/המכשיר כאשר זה אחד מהמקומות הנתמכים שלנו
4. חזור אל `en`

## בסיס המקור

- [אינדקס מיומנות אנגלית של EF 2025](https://www.ef.edu/epi/) לשליטה באנגלית ברמת הארץ
- [מפקד סינגפור 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) עבור סינגפור, שם אנגלית הייתה שפת הבית הנפוצה ביותר עבור 48.3% מהתושבים בגילאי 5+
