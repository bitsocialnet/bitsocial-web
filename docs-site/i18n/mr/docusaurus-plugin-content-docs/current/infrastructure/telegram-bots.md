# टेलिग्राम बॉट्स

बिटसोशियल टेलिग्राम बॉट्स बिटसोशियल नेटवर्कवरील क्लायंट समुदाय सूचीचे निरीक्षण करतात आणि स्वयंचलितपणे नवीन पोस्ट टेलीग्राम चॅनेलमध्ये फॉरवर्ड करतात. प्रत्येक फॉरवर्ड केलेल्या मेसेजमध्ये इनलाइन बटणे समाविष्ट असतात जी 5chan आणि Seedit वरील मूळ पोस्टशी परत लिंक करतात.

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## उपलब्ध बॉट्स

| बॉट          | स्थिती  | वर्णन                                                                          |
| ------------ | ------- | ------------------------------------------------------------------------------ |
| **5चॅन फीड** | सक्रिय  | सर्व 5chan डिरेक्टरीजचे निरीक्षण करते आणि टेलीग्रामवर नवीन पोस्ट फॉरवर्ड करते. |
| **बीज फीड**  | नियोजित | Seedit समुदायांसाठी समान कार्यक्षमता प्रदान करेल.                              |

## सेटअप

### पूर्वआवश्यकता

- Node.js
- यार्न
- टेलीग्राम बॉट टोकन ([बॉटफादर](https://t.me/BotFather))

### स्थापना

रेपॉजिटरी क्लोन करा आणि अवलंबन स्थापित करा:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### कॉन्फिगरेशन

तुमच्या बॉट टोकनसह प्रोजेक्ट रूटमध्ये `.env` फाइल तयार करा:

```env
BOT_TOKEN=your_telegram_bot_token
```

### चालत आहे

तुमचे वातावरण कॉन्फिगर केल्यानंतर बॉट सुरू करा:

```bash
yarn start
```

## पोस्ट फॉरमॅट

जेव्हा बॉट टेलीग्रामवर पोस्ट फॉरवर्ड करतो, तेव्हा त्यात दोन इनलाइन बटणे समाविष्ट असतात:

- **5chan वर पहा** -- 5chan वेब क्लायंटमध्ये पोस्ट उघडते.
- **वेब पोस्ट पहा ** संपादनात पहा. क्लायंट.

हे टेलीग्राम सदस्यांना त्यांच्या पसंतीच्या क्लायंटच्या पूर्ण चर्चेच्या थ्रेडवर थेट जाऊ देते.
