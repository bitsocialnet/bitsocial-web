# seedit

सीडीट हा बिटसोशियल प्रोटोकॉलवर तयार केलेला जुन्या-शैलीच्या रेडडिटचा सर्व्हरलेस, ॲडमिनलेस आणि पूर्णपणे विकेंद्रित पर्याय आहे. हे वापरकर्त्यांना त्यांच्या समुदायांवर संपूर्ण मालकी देते — कोणीही एक तयार करू शकतो आणि एकाधिक समुदाय डीफॉल्ट सूची स्लॉटसाठी स्पर्धा करू शकतात.

## डाउनलोड

| प्लॅटफॉर्म | लिंक                                                    |
| ---------- | ------------------------------------------------------- |
| वेब        | [seedit.app](https://seedit.app)                        |
| वेब (ENS)  | `seedit.eth` ब्रेव्ह ब्राउझर किंवा IPFS कंपेनियन द्वारे |
| डेस्कटॉप   | Mac, Windows आणि Linux साठी उपलब्ध                      |
| मोबाईल     | Android साठी उपलब्ध                                     |

## समुदाय कसे कार्य करतात

seedit subreddits प्रमाणेच समुदायांमध्ये सामग्री आयोजित करते. प्रत्येक समुदाय त्याच्या निर्मात्याच्या मालकीचा असतो आणि केंद्रीकृत प्रशासनाशिवाय कार्य करतो. जेव्हा अनेक समुदाय समान डीफॉल्ट स्लॉटला लक्ष्य करतात, तेव्हा ते त्या स्थानासाठी स्पर्धा करू शकतात.

### समुदाय तयार करणे

नवीन समुदाय तयार करण्याचे दोन मार्ग आहेत:

1. **डेस्कटॉप क्लायंट** — तुमचा समुदाय सेट अप आणि कॉन्फिगर करण्यासाठी डेस्कटॉप ऍप्लिकेशनमध्ये अंगभूत GUI वापरा.
2. **bitsocial-cli** — कमांड लाइनवरून समुदाय तयार आणि व्यवस्थापित करण्यासाठी CLI टूल चालवा.

### डीफॉल्ट समुदाय सूची

मुख्य पानावर दिसणारी डीफॉल्ट समुदायांची यादी यादी भांडारात `default-multisub.json` मध्ये परिभाषित केली आहे. तुमचा समुदाय डीफॉल्टमध्ये जोडण्यासाठी, तुमच्या समुदायाच्या तपशिलांसह एक पुल विनंती सबमिट करा.
