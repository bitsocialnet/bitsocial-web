# व्हाउचर चॅलेंज

व्हाउचर चॅलेंज ही स्पॅम-विरोधी यंत्रणा आहे जी अनन्य व्हाउचर कोडच्या मागे आशयाचे प्रकाशन करते. स्वयंचलित शोधावर विसंबून राहण्याऐवजी, तो समुदाय मालकावर विश्वास हलवतो, जो त्यांचा विश्वास असलेल्या लोकांना कोड मॅन्युअली वितरीत करतो.

**स्रोत कोड:** [github.com/bitsocialnet/voucher-challenge](https://github.com/bitsocialnet/voucher-challenge)

## हे कसे कार्य करते

1. अनन्य समुदायाचे मालक v. कोड.
2. मालक विश्वासू लेखकांना त्यांच्या आवडीच्या चॅनेलद्वारे ते कोड वितरित करतो (थेट संदेश, ईमेल, वैयक्तिक इ.).
3. जेव्हा लेखक प्रकाशित करण्याचा प्रयत्न करतो, तेव्हा आव्हान प्रणाली त्यांना व्हाउचर कोडसाठी सूचित करते.
4. कोड प्रमाणित केला गेला आहे -- जर तो सार्वजनिक केला गेला नसेल, तर तो वापरला गेला नसेल. स्वीकारले.

प्रत्येक व्हाउचर कोड एका विशिष्ट लेखकाशी जोडला जातो एकदा रिडीम केल्यावर, इतरांद्वारे पुन्हा वापरण्यापासून प्रतिबंधित करते.

## ते कधी वापरायचे

व्हाउचर चॅलेंज यासाठी सर्वात योग्य आहे:

- **फक्त-आमंत्रित समुदाय** जेथे सदस्यत्व हेतुपुरस्सर प्रतिबंधित आहे.
- प्रत्येक जागा मालकाने **जिथे वैयक्तिकरित्या / ** zxqsep \*\* zxqsep/> सहभागी.
- **उच्च-विश्वास वातावरण** जेथे स्वयंचलित स्पॅम स्कोअरिंग अनावश्यक किंवा अवांछित आहे.

त्याला मॅन्युअल कोड वितरण आवश्यक असल्याने, ते मोठ्या खुल्या समुदायांमध्ये मोजत नाही, [स्पॅम ब्लॉक](ब्लॉकर-ब्लॉक) चॅलेंज](./evm-contract-call.md) त्याऐवजी.

## एकीकरण

Bitsocial इकोसिस्टममधील इतर अँटी-स्पॅम पॅकेजेसद्वारे वापरल्या जाणाऱ्या त्याच आव्हान इंटरफेसमध्ये व्हाउचर चॅलेंज प्लग इन करते, जेव्हा समुदाय मालक त्यांच्या समुदाय सेटिंग्जद्वारे आव्हान सादर करण्याचा प्रयत्न करतात तेव्हा ते स्वयंचलितपणे सक्षम करतात.
