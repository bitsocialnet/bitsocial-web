# i18n 默认语言策略

匿名访问者现在按以下顺序解析语言：

1. `?lang=` 查询参数
2. 从 `localStorage` 保存的语言选择器选择
3. 区域感知匿名默认值：
   - 在明确列出的英语密集地区强制使用英语
   - 否则，当浏览器/设备语言是我们支持的区域设置之一时，请使用它
4. 回退到 `en`

## 源基线

- [2025年EF英语掌握度指数](https://www.ef.edu/epi/) 国家级英语水平
- [2020年新加坡人口普查](https://www.singstat.gov.sg/modules/infographics/census-2020) 新加坡，英语是 48.3% 5 岁以上居民最常用的家庭语言
