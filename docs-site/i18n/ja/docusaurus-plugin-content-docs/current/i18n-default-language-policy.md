# i18n デフォルト言語ポリシー

匿名の訪問者は次の順序で言語を解決するようになりました:

1. `?lang=` クエリ パラメータ
2. `localStorage` からの保存された言語セレクターの選択
3. 地域を意識した匿名のデフォルト:
   - 明示的にリストされている英語が多い言語では英語を強制します地域
   - それ以外の場合、サポートされているロケールの場合はブラウザ/デバイスの言語を使用します
4. `en` に戻ります

## ソース ベースライン

- [EF英語能力指数 2025](国レベルの英語能力については https://www.ef.edu/epi/)
- [2020]年シンガポール国勢調査](シンガポールの https://www.singstat.gov.sg/modules/infographics/census-2020) では、5 歳以上の住民の 48.3% にとって英語が最も頻繁に話される家庭言語でした
