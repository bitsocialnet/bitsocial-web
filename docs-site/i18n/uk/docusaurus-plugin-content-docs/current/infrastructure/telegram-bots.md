# Боти Telegram

Боти Bitsocial Telegram відстежують списки спільнот клієнтів у мережі Bitsocial і автоматично пересилають нові повідомлення на канали Telegram. Кожне переслане повідомлення містить вбудовані кнопки, які посилаються на вихідну публікацію на 5chan і Seedit.

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## Доступні боти

| Бот               | Статус     | Опис                                                                 |
| ----------------- | ---------- | -------------------------------------------------------------------- |
| **Стрічка 5chan** | Активний   | Відстежує всі каталоги 5chan і пересилає нові публікації в Telegram. |
| **Seedit Feed**   | Планується | Забезпечить ту саму функціональність для спільнот Seedit.            |

## Налаштування

### передумови

- Node.js
- Пряжа
- Маркер бота Telegram (створіть його за допомогою [BotFather](https://t.me/BotFather)))

### монтаж

Клонуйте репозиторій і встановіть залежності:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### Конфігурація

Створіть файл `.env` у корені проекту з вашим маркером бота:

```env
BOT_TOKEN=your_telegram_bot_token
```

### Біг

Запустіть бота після налаштування середовища:

```bash
yarn start
```

## Формат публікації

Коли бот пересилає публікацію в Telegram, він містить дві вбудовані кнопки:

- **Переглянути на 5chan** -- Відкриває публікацію у веб-клієнті 5chan.
- **Переглянути на Seedit** -- Відкриває публікацію у веб-клієнті Seedit.

Це дозволяє передплатникам Telegram переходити безпосередньо до повної теми обговорення будь-якого клієнта, який вони віддають перевагу.
