# seedit

seedit — це повністю децентралізована альтернатива старого Reddit без сервера, адміністратора, побудована на протоколі Bitsocial. Це дає користувачам повне право власності на свої спільноти — будь-хто може створити одну, і кілька спільнот можуть конкурувати за стандартні слоти в списку.

## Завантаження

| Платформа    | Посилання                                           |
| ------------ | --------------------------------------------------- |
| Веб          | [seedit.app](https://seedit.app)                    |
| Веб (ENS)    | `seedit.eth` через браузер Brave або IPFS Companion |
| Робочий стіл | Доступно для Mac, Windows і Linux                   |
| Мобільний    | Доступно для Android                                |

## Як працюють спільноти

seedit організовує вміст у спільнотах, подібних до субредітів. Кожна спільнота належить її творцю та діє без централізованого адміністрування. Коли кілька спільнот націлені на те саме місце за замовчуванням, вони можуть конкурувати за цю позицію.

### Створення спільноти

Створити нову спільноту можна двома способами:

1. **Настільний клієнт** — використовуйте вбудований графічний інтерфейс програми для настільного комп’ютера, щоб налаштувати та налаштувати свою спільноту.
2. **bitsocial-cli** — запустіть інструмент CLI для створення та керування спільнотами з командного рядка.

### Список спільноти за умовчанням

Список спільнот за замовчуванням, які з’являються на першій сторінці, визначено в `default-multisub.json` у сховищі списків. Щоб додати свою спільноту до стандартних, надішліть запит на отримання інформації про свою спільноту.
