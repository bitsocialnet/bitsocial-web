# Bitsocial мережа

Bitsocial Network — це назва, яка використовується в генеральному плані для спільного рівня додатків, який розміщуватиметься під програмами Bitsocial.

Ідея полягає не в тому, щоб замінити соціальний прошарок «рівний-рівному». Це забезпечення спільних економічних і координаційних примітивів, які знадобляться соціальним програмам, якщо вони збираються конкурувати з існуючими платформами в масштабі.

## Що це означає для влади

- децентралізовані домени Bitsocial, такі як `.bso`
- спільна ліквідність між програмами
- міцні рейки монетизації
- фінансові структури, які важче задушити банкам або платформам
- мережеві ефекти, які не залежать від того, що одна компанія володіє всім стеком

## Чому це важливо

Децентралізація спільнот та ідентичностей є необхідною, але недостатньо децентралізувати всі соціальні мережі.

Якщо економічний рівень все ще залежить від кількох централізованих рейок, то на додатки легко тиснути, деплатформувати чи фінансово голодувати. Bitsocial Network є запропонованою відповіддю на другий рівень залежності.

## Відносини до решти Bitsocial

Bitsocial Network пропонується як спільна інфраструктура для програм Bitsocial, а не як продукт, з яким користувачі взаємодіють безпосередньо щодня.

Результатом для громадськості буде:

- спільноти залишаються рівноправними
- програми залишаються диференційованими
- монетизація та неймінг стають довговічнішими
- цінність може переміщуватися в екосистемі без відновлення централізованого власника платформи

## Чому це відноситься до Фази 4

Генеральний план розміщує це після попередніх перевірочних пунктів на рівні програми:

1. децентралізувати іміджборди
2. запустіть Bitsocial RPC
3. децентралізувати форуми
4. запустіть Bitsocial Network

Така послідовність має сенс, оскільки координаційний рівень на рівні екосистеми більш корисний, якщо в мережі вже існують реальні програми та реальна поведінка користувачів.
