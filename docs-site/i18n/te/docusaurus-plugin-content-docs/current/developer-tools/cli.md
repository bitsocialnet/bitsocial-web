# CLI

:::warning Legacy Naming
ఈ ప్యాకేజీ ప్రస్తుతం దాని అప్‌స్ట్రీమ్ డిపెండెన్సీ నుండి వారసత్వంగా పొందిన లెగసీ నేమింగ్ కన్వెన్షన్‌లను ఉపయోగిస్తోంది. కమాండ్‌లు, అవుట్‌పుట్ మరియు కాన్ఫిగరేషన్‌లో "plebbit"కి సంబంధించిన సూచనలు భవిష్యత్ విడుదలలో "bitsocial"కి తరలించబడతాయి. కార్యాచరణ ప్రభావితం కాదు.
:::

`bitsocial-cli` అనేది Bitsocial ప్రోటోకాల్ బ్యాకెండ్‌తో పరస్పర చర్య చేయడానికి కమాండ్-లైన్ సాధనం. ఇది స్థానిక P2P డెమోన్‌ని అమలు చేయడానికి, సంఘాలను సృష్టించడానికి మరియు కాన్ఫిగర్ చేయడానికి మరియు కంటెంట్‌ను ప్రచురించడానికి మిమ్మల్ని అనుమతిస్తుంది -- అన్నీ టెర్మినల్ నుండి.

ఇది `plebbit-js` పైన నిర్మించబడింది మరియు కమ్యూనిటీ సృష్టి మరియు నోడ్ నిర్వహణ కోసం [5chan](/apps/5chan/) మరియు [సీడిట్](/apps/seedit/) ద్వారా ఉపయోగించబడుతుంది.

## సంస్థాపన

Windows, macOS మరియు Linux కోసం ముందుగా నిర్మించిన బైనరీలు అందుబాటులో ఉన్నాయి. GitHub నుండి మీ ప్లాట్‌ఫారమ్ కోసం తాజా విడుదలను డౌన్‌లోడ్ చేయండి:

**[GitHub విడుదల నుండి డౌన్‌లోడ్ చేయండి](https://github.com/bitsocialnet/bitsocial-cli/releases)**

డౌన్‌లోడ్ చేసిన తర్వాత, బైనరీని ఎక్జిక్యూటబుల్ చేయండి (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## డెమోన్‌ను నడుపుతోంది

CLI యొక్క అత్యంత సాధారణ ఉపయోగం Bitsocial నోడ్‌ను అమలు చేయడం. డెమోన్ P2P నెట్‌వర్కింగ్ లేయర్‌ను ప్రారంభిస్తుంది మరియు క్లయింట్‌లు కనెక్ట్ చేయగల స్థానిక APIని బహిర్గతం చేస్తుంది.

```bash
bitsocial-cli daemon
```

మొదటి ప్రయోగంలో, డెమోన్ మీ నోడ్, కమ్యూనిటీలు మరియు సెట్టింగ్‌లను నిర్వహించడానికి బ్రౌజర్ ఆధారిత గ్రాఫికల్ ఇంటర్‌ఫేస్ అయిన **WebUI**కి లింక్‌లను అందిస్తుంది. మీరు టెర్మినల్ ఆదేశాల కంటే GUIని ఇష్టపడితే ఇది ఉపయోగపడుతుంది.

## కీ ఆదేశాలు

| ఆదేశం               | వివరణ                                                    |
| ------------------- | -------------------------------------------------------- |
| `daemon`            | Bitsocial P2P నోడ్‌ను ప్రారంభించండి                      |
| `create subplebbit` | కొత్త సంఘాన్ని సృష్టించండి                               |
| `subplebbit edit`   | సంఘం సెట్టింగ్‌లను నవీకరించండి (శీర్షిక, వివరణ, నియమాలు) |
| `subplebbit list`   | ఈ నోడ్‌లో హోస్ట్ చేయబడిన సంఘాల జాబితా                    |
| `subplebbit start`  | నిర్దిష్ట సంఘానికి సేవ చేయడం ప్రారంభించండి               |
| `subplebbit stop`   | నిర్దిష్ట సంఘానికి సేవ చేయడం ఆపు                         |

అందుబాటులో ఉన్న ఎంపికలు మరియు ఫ్లాగ్‌లను చూడటానికి `--help`తో ఏదైనా ఆదేశాన్ని అమలు చేయండి:

```bash
bitsocial-cli daemon --help
bitsocial-cli create subplebbit --help
```

## సాధారణ వర్క్‌ఫ్లో

కొత్త కమ్యూనిటీని హోస్ట్ చేయడానికి ఒక సాధారణ సెటప్ ఫ్లో:

```bash
# 1. Start the daemon
bitsocial-cli daemon

# 2. In another terminal, create a community
bitsocial-cli create subplebbit

# 3. Configure the community
bitsocial-cli subplebbit edit <address> --title "My Community" --description "A decentralized forum"

# 4. Start serving it
bitsocial-cli subplebbit start <address>
```

సంఘం ఇప్పుడు Bitsocial నెట్‌వర్క్‌లో ప్రత్యక్ష ప్రసారం చేయబడుతోంది మరియు ఏదైనా అనుకూల క్లయింట్ నుండి ప్రాప్యత చేయగలదు.
