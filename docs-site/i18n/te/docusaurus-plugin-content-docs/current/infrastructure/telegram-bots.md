# టెలిగ్రామ్ బాట్‌లు

Bitsocial టెలిగ్రామ్ బాట్‌లు Bitsocial నెట్‌వర్క్‌లోని క్లయింట్ కమ్యూనిటీ జాబితాలను పర్యవేక్షిస్తాయి మరియు కొత్త పోస్ట్‌లను టెలిగ్రామ్ ఛానెల్‌లలోకి స్వయంచాలకంగా ఫార్వార్డ్ చేస్తాయి. ఫార్వార్డ్ చేయబడిన ప్రతి సందేశం 5chan మరియు సీడిట్‌లోని అసలు పోస్ట్‌కి తిరిగి లింక్ చేసే ఇన్‌లైన్ బటన్‌లను కలిగి ఉంటుంది.

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## అందుబాటులో ఉన్న బాట్‌లు

| బాట్            | స్థితి   | వివరణ                                                                                                 |
| --------------- | -------- | ----------------------------------------------------------------------------------------------------- |
| **5chan Feed**  | యాక్టివ్ | అన్ని 5chan డైరెక్టరీలను పర్యవేక్షిస్తుంది మరియు కొత్త పోస్ట్‌లను టెలిగ్రామ్‌కు ఫార్వార్డ్ చేస్తుంది. |
| **సీడిట్ ఫీడ్** | ప్లాన్డ్ | Seedit కమ్యూనిటీలకు అదే కార్యాచరణను అందిస్తుంది.                                                      |

## సెటప్

### ముందస్తు అవసరాలు

- Node.js
- నూలు
- ఒక టెలిగ్రామ్ బాట్ టోకెన్ ([బోట్ ఫాదర్](https://t.me/BotFather)) ద్వారా ఒకదాన్ని సృష్టించండి

### సంస్థాపన

రిపోజిటరీని క్లోన్ చేసి డిపెండెన్సీలను ఇన్‌స్టాల్ చేయండి:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### ఆకృతీకరణ

మీ బోట్ టోకెన్‌తో ప్రాజెక్ట్ రూట్‌లో `.env` ఫైల్‌ను సృష్టించండి:

```env
BOT_TOKEN=your_telegram_bot_token
```

### నడుస్తోంది

మీ పర్యావరణాన్ని కాన్ఫిగర్ చేసిన తర్వాత బాట్‌ను ప్రారంభించండి:

```bash
yarn start
```

## పోస్ట్ ఫార్మాట్

బోట్ ఒక పోస్ట్‌ను టెలిగ్రామ్‌కి ఫార్వార్డ్ చేసినప్పుడు, అది రెండు ఇన్‌లైన్ బటన్‌లను కలిగి ఉంటుంది:

- **5chanలో వీక్షించండి** -- 5chan వెబ్ క్లయింట్‌లో పోస్ట్‌ను తెరుస్తుంది.
- **సీడిట్‌లో వీక్షించండి** -- సీడిట్ వెబ్ క్లయింట్‌లో పోస్ట్‌ను తెరుస్తుంది.

ఇది టెలిగ్రామ్ సబ్‌స్క్రైబర్‌లను వారు ఇష్టపడే క్లయింట్‌పై పూర్తి చర్చా థ్రెడ్‌కి నేరుగా వెళ్లేలా చేస్తుంది.
