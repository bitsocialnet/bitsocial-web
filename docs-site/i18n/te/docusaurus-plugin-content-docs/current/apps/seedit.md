# విత్తనం

seedit అనేది Bitsocial ప్రోటోకాల్‌పై నిర్మించబడిన పాత-శైలి Redditకి సర్వర్‌లెస్, అడ్మిన్‌లెస్ మరియు పూర్తిగా వికేంద్రీకృత ప్రత్యామ్నాయం. ఇది వినియోగదారులకు వారి సంఘాలపై పూర్తి యాజమాన్యాన్ని ఇస్తుంది - ఎవరైనా ఒకదాన్ని సృష్టించవచ్చు మరియు డిఫాల్ట్ జాబితా స్లాట్‌ల కోసం బహుళ సంఘాలు పోటీపడవచ్చు.

## డౌన్‌లోడ్‌లు

| వేదిక       | లింక్                                                  |
| ----------- | ------------------------------------------------------ | -------------------- |
| వెబ్        | [seedit.app](https://seedit.app)                       |
| వెబ్ (ENS)  | బ్రేవ్ బ్రౌజర్ లేదా IPFS కంపానియన్ ద్వారా `seedit.eth` |
| డెస్క్‌టాప్ | Mac, Windows మరియు Linux కోసం అందుబాటులో ఉంది          |
| మొబైల్      | Android                                                | కోసం అందుబాటులో ఉంది |

## సంఘాలు ఎలా పని చేస్తాయి

seedit సబ్‌రెడిట్‌ల మాదిరిగానే కమ్యూనిటీలుగా కంటెంట్‌ను నిర్వహిస్తుంది. ప్రతి సంఘం దాని సృష్టికర్త స్వంతం మరియు కేంద్రీకృత పరిపాలన లేకుండా పనిచేస్తుంది. బహుళ సంఘాలు ఒకే డిఫాల్ట్ స్లాట్‌ను లక్ష్యంగా చేసుకున్నప్పుడు, వారు ఆ స్థానం కోసం పోటీ పడవచ్చు.

### సంఘాన్ని సృష్టిస్తోంది

కొత్త సంఘాన్ని సృష్టించడానికి రెండు మార్గాలు ఉన్నాయి:

1. **డెస్క్‌టాప్ క్లయింట్** — మీ సంఘాన్ని సెటప్ చేయడానికి మరియు కాన్ఫిగర్ చేయడానికి డెస్క్‌టాప్ అప్లికేషన్‌లో అంతర్నిర్మిత GUIని ఉపయోగించండి.
2. **bitsocial-cli** — కమాండ్ లైన్ నుండి కమ్యూనిటీలను సృష్టించడానికి మరియు నిర్వహించడానికి CLI సాధనాన్ని అమలు చేయండి.

### డిఫాల్ట్ సంఘం జాబితా

మొదటి పేజీలో కనిపించే డిఫాల్ట్ కమ్యూనిటీల జాబితా జాబితాల రిపోజిటరీలోని `default-multisub.json`లో నిర్వచించబడింది. మీ సంఘాన్ని డిఫాల్ట్‌లకు జోడించడానికి, మీ సంఘం వివరాలతో పుల్ అభ్యర్థనను సమర్పించండి.
