# 5చాన్

5chan అనేది సర్వర్‌లెస్, అడ్మిన్‌లెస్ మరియు పూర్తిగా వికేంద్రీకరించబడిన ఇమేజ్‌బోర్డ్, ఇది బిట్సోషియల్ ప్రోటోకాల్‌పై నడుస్తుంది. వికేంద్రీకృత యాజమాన్యాన్ని పరిచయం చేస్తున్నప్పుడు ఇది సుపరిచితమైన ఇమేజ్‌బోర్డ్ డైరెక్టరీ నిర్మాణాన్ని అనుసరిస్తుంది - ఎవరైనా బోర్డుని సృష్టించవచ్చు మరియు ఓటింగ్ విధానం ద్వారా ఒకే డైరెక్టరీ స్లాట్ కోసం బహుళ బోర్డులు పోటీపడవచ్చు.

| మొబైల్      | Android                                       | కోసం అందుబాటులో ఉంది |

## బోర్డులు ఎలా పని చేస్తాయి

5chan క్లాసిక్ డైరెక్టరీ లేఅవుట్ (ఉదా., `/b/`, `/g/`) ఉపయోగించి కంటెంట్‌ను బోర్డులుగా నిర్వహిస్తుంది. సెంట్రల్ అడ్మిన్ ప్రతి బోర్డ్‌ను నియంత్రించే సాంప్రదాయ ఇమేజ్‌బోర్డ్‌ల వలె కాకుండా, 5chan ఏ వినియోగదారునైనా వారి స్వంత బోర్డ్‌ను సృష్టించడానికి మరియు పూర్తిగా స్వంతం చేసుకోవడానికి అనుమతిస్తుంది. బహుళ బోర్డులు ఒకే డైరెక్టరీ స్లాట్‌ను లక్ష్యంగా చేసుకున్నప్పుడు, అవి ఓటింగ్ ద్వారా ఆ స్థానం కోసం పోటీపడతాయి.

### ఒక బోర్డు సృష్టిస్తోంది

కొత్త బోర్డుని సృష్టించడానికి, మీరు `bitsocial-cli`ని పీర్-టు-పీర్ నోడ్‌గా అమలు చేయాలి. ఇది మీ బోర్డు ఏ కేంద్ర సర్వర్‌పై ఆధారపడకుండా వికేంద్రీకృత పద్ధతిలో హోస్ట్ చేయబడిందని నిర్ధారిస్తుంది.

### డైరెక్టరీ కేటాయింపులు

డైరెక్టరీ స్లాట్ అసైన్‌మెంట్‌లు (ఏ మార్గంలో ఏ బోర్డు కనిపిస్తుంది) ప్రస్తుతం `5chan-directories.json` ఫైల్‌కి GitHub పుల్ అభ్యర్థనల ద్వారా నిర్వహించబడుతుంది. ఇది తాత్కాలిక ప్రక్రియ — భవిష్యత్ విడుదలలు డైరెక్టరీ అసైన్‌మెంట్‌లను స్వయంచాలకంగా నిర్వహించడానికి యాప్‌లో బోర్డు సృష్టి మరియు పబ్‌సబ్ ఆధారిత ఓటింగ్‌కు మద్దతు ఇస్తాయి.

## అంతర్గతాలు

హుడ్ కింద, 5chan దాని ప్రోటోకాల్ పరస్పర చర్యల కోసం plebbit-js API లేయర్‌ని ఉపయోగిస్తుంది. ఎగువ హెచ్చరికలో గుర్తించినట్లుగా, ఈ అంతర్గత సూచనలు ఇప్పటికీ బిట్సోషల్ రీబ్రాండ్‌కు పూర్వం ఉన్న లెగసీ పేరును కలిగి ఉన్నాయి.
