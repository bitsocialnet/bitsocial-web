# टेलीग्राम बॉट

बिटसोशल टेलीग्राम बॉट बिटसोशल नेटवर्क पर क्लाइंट समुदाय सूचियों की निगरानी करते हैं और टेलीग्राम चैनलों में स्वचालित रूप से नए पोस्ट अग्रेषित करते हैं। प्रत्येक अग्रेषित संदेश में इनलाइन बटन शामिल होते हैं जो 5chan और Seedit पर मूल पोस्ट से लिंक होते हैं।

- **GitHub**: [बिटसोशलनेट/बिटसोशल-टेलीग्राम-बॉट्स](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## उपलब्ध बॉट

| बॉट            | स्थिति  | विवरण                                                                                |
| -------------- | ------- | ------------------------------------------------------------------------------------ |
| **5चान फ़ीड**  | सक्रिय  | सभी 5chan निर्देशिकाओं पर नज़र रखता है और नए पोस्ट को टेलीग्राम पर अग्रेषित करता है। |
| **सीडिट फ़ीड** | नियोजित | सीडिट समुदायों के लिए समान कार्यक्षमता प्रदान करेगा।                                 |

## सेटअप

### पूर्वावश्यकताएँ

- नोड.जे.एस
- सूत
- एक टेलीग्राम बॉट टोकन ([बोटफाडर](https://t.me/BotFather)) के माध्यम से एक बनाएं)

### स्थापना

रिपॉजिटरी को क्लोन करें और निर्भरताएँ स्थापित करें:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### विन्यास

अपने बॉट टोकन के साथ प्रोजेक्ट रूट में एक `.env` फ़ाइल बनाएं:

```env
BOT_TOKEN=your_telegram_bot_token
```

### चल रहा है

अपना परिवेश कॉन्फ़िगर करने के बाद बॉट प्रारंभ करें:

```bash
yarn start
```

## पोस्ट प्रारूप

जब बॉट किसी पोस्ट को टेलीग्राम पर अग्रेषित करता है, तो इसमें दो इनलाइन बटन शामिल होते हैं:

- **5chan पर देखें** -- पोस्ट को 5chan वेब क्लाइंट में खोलता है।
- **सीडिट पर देखें** -- सीडिट वेब क्लाइंट में पोस्ट खोलता है।

इससे टेलीग्राम सब्सक्राइबर्स सीधे अपनी पसंद के किसी भी क्लाइंट पर पूर्ण चर्चा थ्रेड पर जा सकते हैं।
