# कैप्चा कैनवास चैलेंज

:::warning Legacy Naming
यह पैकेज मूल रूप से `@plebbit` दायरे के अंतर्गत प्रकाशित किया गया था। इसका नाम बदलकर `@bitsocial/captcha-canvas-challenge` कर दिया गया है। पुराने नाम का संदर्भ अभी भी पुराने दस्तावेज़ों या कोडबेस में दिखाई दे सकता है।
:::

कैप्चा कैनवस चैलेंज एक स्टैंडअलोन इमेज कैप्चा जनरेटर है जो मूल रूप से `plebbit-js` से निकाला गया है। यह यादृच्छिक पाठ को HTML कैनवास पर प्रस्तुत करता है और परिणामी छवि लौटाता है, जिसे समुदाय लेखकों के सामने स्पैम चुनौती के रूप में प्रस्तुत कर सकते हैं।

**स्रोत कोड:** [github.com/bitsocialnet/captcha-canvas-challenge](https://github.com/bitsocialnet/captcha-canvas-challenge)

## आवश्यकताएँ

- **Node.js** >= 22
- **केवल ईएसएम** -- यह पैकेज CommonJS बिल्ड को शिप नहीं करता है।
- **रनटाइम सहकर्मी निर्भरता:** `@plebbit/plebbit-js` (`@pkc/pkc-js` पर माइग्रेट करना)

## स्थापना

```bash
npm install @bitsocial/captcha-canvas-challenge
```

## कॉन्फ़िगरेशन विकल्प

| विकल्प       | प्रकार   | डिफ़ॉल्ट  | विवरण                                                     |
| ------------ | -------- | --------- | --------------------------------------------------------- |
| `characters` | `number` | `6`       | कैप्चा छवि में प्रस्तुत यादृच्छिक वर्णों की संख्या।       |
| `height`     | `number` | `100`     | उत्पन्न छवि की ऊंचाई पिक्सेल में.                         |
| `width`      | `number` | `300`     | उत्पन्न छवि की चौड़ाई पिक्सेल में.                        |
| `colors`     | `string` | `#32cf7e` | कैप्चा टेक्स्ट के लिए प्राथमिक रंग का उपयोग किया जाता है। |

## यह कैसे काम करता है

1. जनरेटर कॉन्फ़िगर लंबाई की एक यादृच्छिक स्ट्रिंग चुनता है।
2. ओसीआर का विरोध करने के लिए स्ट्रिंग को दृश्य शोर के साथ एक कैनवास पर प्रस्तुत किया जाता है।
3. परिणामी छवि (और अपेक्षित उत्तर) लौटा दी जाती है ताकि कॉलिंग एप्लिकेशन चुनौती पेश कर सके और बाद में प्रतिक्रिया को सत्यापित कर सके।

क्योंकि पैकेज एक शुद्ध छवि जनरेटर है, यह नेटवर्किंग या सत्र प्रबंधन को स्वयं संभाल नहीं पाता है। इसे एक बड़े चुनौती प्रवाह में एकीकृत करने का इरादा है - उदाहरण के लिए, [स्पैम ब्लॉकर](./स्पैम-ब्लॉकर.एमडी) द्वारा समर्थित चुनौती प्रकारों में से एक के रूप में।
