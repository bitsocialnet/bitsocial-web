# خط مشی زبان پیش فرض i18n

بازدیدکنندگان ناشناس اکنون زبان را به ترتیب زیر حل می کنند:

1. پارامتر پرس و جو `?lang=`
2. انتخاب انتخابگر زبان از `localStorage` ذخیره شده است
3. پیش‌فرض ناشناس آگاه از منطقه:
   - زبان انگلیسی را در مناطقی که به صراحت فهرست شده اند، تحمیل کنید
   - در غیر این صورت از زبان مرورگر/دستگاه زمانی که یکی از زبان های پشتیبانی شده ما است استفاده کنید
4. به `en` برگردید

## مبنا منبع

- [شاخص مهارت انگلیسی EF 2025](https://www.ef.edu/epi/) for country-level proficiency English
- [سرشماری سنگاپور 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) برای سنگاپور، جایی که انگلیسی بیشترین صحبت را در زبان خانه برای 48.3 درصد از ساکنان بالای 5 سال داشت.
