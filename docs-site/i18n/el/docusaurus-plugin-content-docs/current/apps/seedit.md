# sedit

Το seedit είναι μια εναλλακτική λύση χωρίς διακομιστή, χωρίς διαχειριστή και πλήρως αποκεντρωμένη στο παλιού τύπου Reddit, που βασίζεται στο πρωτόκολλο Bitsocial. Παρέχει στους χρήστες πλήρη ιδιοκτησία των κοινοτήτων τους — ο καθένας μπορεί να δημιουργήσει μία και πολλές κοινότητες μπορούν να ανταγωνιστούν για προεπιλεγμένες θέσεις υποδοχής καταχώρισης.

## Πώς λειτουργούν οι κοινότητες

Το sedit οργανώνει το περιεχόμενο σε κοινότητες παρόμοιες με subreddits. Κάθε κοινότητα ανήκει στον δημιουργό της και λειτουργεί χωρίς κεντρική διαχείριση. Όταν πολλές κοινότητες στοχεύουν την ίδια προεπιλεγμένη υποδοχή, μπορούν να ανταγωνιστούν για αυτήν τη θέση.

### Δημιουργία κοινότητας

Υπάρχουν δύο τρόποι για να δημιουργήσετε μια νέα κοινότητα:

1. **Πελάτης επιφάνειας εργασίας** — Χρησιμοποιήστε το ενσωματωμένο GUI στην εφαρμογή επιφάνειας εργασίας για να ρυθμίσετε και να διαμορφώσετε την κοινότητά σας.
2. **bitsocial-cli** — Εκτελέστε το εργαλείο CLI για να δημιουργήσετε και να διαχειριστείτε κοινότητες από τη γραμμή εντολών.

### Προεπιλεγμένη λίστα κοινότητας

Η λίστα των προεπιλεγμένων κοινοτήτων που εμφανίζονται στην πρώτη σελίδα ορίζεται στο `default-multisub.json` εντός του αποθετηρίου λιστών. Για να προσθέσετε την κοινότητά σας στις προεπιλογές, υποβάλετε ένα αίτημα έλξης με τα στοιχεία της κοινότητάς σας.
