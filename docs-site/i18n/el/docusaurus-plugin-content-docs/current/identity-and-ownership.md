# Ταυτότητα και κοινοτική ιδιοκτησία

Τα προφίλ και οι κοινότητες στο Bitsocial ελέγχονται από ζεύγη κλειδιών και όχι από λογαριασμούς πλατφόρμας με δυνατότητα ανάκλησης.

Αυτό αλλάζει τη σχέση μεταξύ ιδιοκτησίας και παρόχων υπηρεσιών. Μπορείτε να αναθέσετε χρόνο λειτουργίας, φιλοξενία ή ευκολία σε άλλη υπηρεσία χωρίς να αναθέσετε σε αυτήν την υπηρεσία μόνιμη επιμέλεια της ίδιας της ταυτότητας ή της κοινότητας.

## Γιατί τα κλειδιά έχουν σημασία εδώ

Ένας λογαριασμός φιλοξενούμενης πλατφόρμας ενοικιάζεται τελικά. Υπάρχει όσο η πλατφόρμα συνεχίζει να το τιμά.

Μια ταυτότητα ελεγχόμενη από κλειδί είναι διαφορετική:

- το ιδιωτικό κλειδί είναι η πηγή της εξουσίας
- η φιλοξενία μπορεί να αντικατασταθεί
- Η επιλογή της εφαρμογής μπορεί να αλλάξει
- η ιδιοκτησία δεν χρειάζεται να κινηθεί με υποδομές

## Τι πρέπει να σημαίνει αντιπροσωπεία

Η ανάθεση θα πρέπει να σημαίνει βοήθεια με τη διαθεσιμότητα, όχι παράδοση ελέγχου.

Για παράδειγμα, ένα δημόσιο RPC ή μια υπηρεσία φιλοξενίας μπορεί να διατηρήσει έναν κόμβο στο διαδίκτυο, αλλά αυτή η υπηρεσία δεν θα πρέπει να χρειάζεται τη δύναμη για να κατακτήσει μόνιμα το προφίλ ή την κοινότητα. Η βοήθεια είναι χρήσιμη. Η επιμέλεια θα πρέπει να παραμείνει προαιρετική.

## Το ευρύτερο σημείο

Αυτό κάνει μια ταυτότητα Bitsocial να αισθάνεται πιο κοντά σε ένα περιουσιακό στοιχείο που ελέγχεται από το πορτοφόλι παρά σε ένα όνομα χρήστη που νοικιάζεται από μια εταιρεία. Το επίπεδο υπηρεσίας μπορεί να αντικατασταθεί. Η σχέση ιδιοκτήτη θα πρέπει να επιβιώσει αυτής της ανταλλαγής.
