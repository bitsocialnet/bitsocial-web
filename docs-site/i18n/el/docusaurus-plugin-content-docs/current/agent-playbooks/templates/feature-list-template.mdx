# Πρότυπο λίστας δυνατοτήτων

Αυτή η σελίδα αντικατοπτρίζει το πρότυπο JSON που αποστέλλεται στο `docs/agent-playbooks/templates/feature-list.template.json`, ώστε να μπορεί να περιηγηθεί απευθείας μέσα στον ιστότοπο των εγγράφων.

```json
{
  "task": "replace-with-task-slug",
  "last_updated": "YYYY-MM-DD",
  "items": [
    {
      "id": "F001",
      "priority": 1,
      "status": "pending",
      "description": "Describe one end-to-end feature or one reviewable task slice.",
      "verification": ["List the command or user-visible check that proves this item works."],
      "files": [],
      "notes": ""
    }
  ]
}
```
