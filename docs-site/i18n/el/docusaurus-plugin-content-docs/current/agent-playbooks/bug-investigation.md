# Ροή εργασιών διερεύνηση σφαλμάτων

Χρησιμοποιήστε το όταν αναφέρεται ένα σφάλμα σε ένα συγκεκριμένο μπλοκ αρχείου/γραμμής/κώδικα.

## Υποχρεωτικό Πρώτο Βήμα

Πριν την επεξεργασία, ελέγξτε το git history για τον σχετικό κώδικα. Προηγούμενοι συνεργάτες μπορεί να έχουν εισαγάγει συμπεριφορά για μια περίπτωση αιχμής/λύση.

## Ροή εργασιών

1. Σάρωση πρόσφατων τίτλων δέσμευσης (μόνο τίτλοι) για το αρχείο/περιοχή:

```bash
# Recent commit titles for a specific file
git log --oneline -10 -- src/components/post-desktop/post-desktop.tsx

# Recent commit titles for a specific line range
git blame -L 120,135 src/components/post-desktop/post-desktop.tsx
```

2. Επιθεωρήστε μόνο σχετικές δεσμεύσεις με διαφορές εύρους:

```bash
# Show commit message + diff for one file
git show <commit-hash> -- path/to/file.tsx
```

3. Συνεχίστε με την αναπαραγωγή και διορθώστε αφού κατανοήσετε το ιστορικό πλαίσιο.

## Κανόνας αντιμετώπισης προβλημάτων

Όταν αποκλείεται, αναζητήστε στον ιστό πρόσφατες διορθώσεις/επιλύσεις.
