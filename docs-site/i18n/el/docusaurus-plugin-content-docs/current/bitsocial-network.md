# Bitsocial Network

Το Bitsocial Network είναι το όνομα που χρησιμοποιείται στο κύριο σχέδιο για το κοινό επίπεδο αλυσίδας εφαρμογών που θα βρίσκεται κάτω από τις εφαρμογές Bitsocial.

Η ιδέα δεν είναι να αντικαταστήσουμε το peer-to-peer κοινωνικό στρώμα. Είναι να παρέχει τα κοινά οικονομικά και συντονιστικά πρωτόγονα που θα χρειαστούν οι κοινωνικές εφαρμογές, εάν πρόκειται να ανταγωνιστούν τις κατεστημένες πλατφόρμες σε κλίμακα.

## Τι σημαίνει η εξουσία

- αποκεντρωμένους τομείς Bitsocial όπως το `.bso`
- κοινή ρευστότητα σε όλες τις εφαρμογές
- ανθεκτικές ράγες δημιουργίας εσόδων
- χρηματοπιστωτικές δομές που είναι πιο δύσκολο για τις τράπεζες ή τις πλατφόρμες να πνιγούν
- εφέ δικτύου που δεν εξαρτώνται από μια εταιρεία που κατέχει ολόκληρη τη στοίβα

## Γιατί έχει σημασία

Η αποκέντρωση των κοινοτήτων και των ταυτοτήτων είναι απαραίτητη, αλλά δεν αρκεί η αποκέντρωση όλων των μέσων κοινωνικής δικτύωσης.

Εάν το οικονομικό επίπεδο εξακολουθεί να εξαρτάται από μια χούφτα κεντρικών σιδηροδρομικών γραμμών, τότε οι εφαρμογές παραμένουν εύκολο να πιεστούν, να αποπλαστοποιηθούν ή να λιμοκτονήσουν οικονομικά. Το Bitsocial Network είναι η προτεινόμενη απάντηση σε αυτό το δεύτερο επίπεδο εξάρτησης.

## Σχέση με το υπόλοιπο Bitsocial

Το Bitsocial Network προτείνεται ως κοινόχρηστη υποδομή για εφαρμογές Bitsocial, όχι ως το προϊόν με το οποίο αλληλεπιδρούν απευθείας οι χρήστες καθημερινά.

Το αποτέλεσμα στο κοινό θα ήταν:

- οι κοινότητες παραμένουν ομότιμες
- οι εφαρμογές παραμένουν διαφοροποιημένες
- η δημιουργία εσόδων και η ονομασία γίνονται πιο ανθεκτικές
- Η αξία μπορεί να μετακινηθεί σε όλο το οικοσύστημα χωρίς να δημιουργήσει εκ νέου έναν κεντρικό ιδιοκτήτη πλατφόρμας

## Γιατί αυτό βρίσκεται στη Φάση 4

Το κύριο σχέδιο το τοποθετεί μετά από τα προηγούμενα σημεία απόδειξης σε επίπεδο εφαρμογής:

1. αποκέντρωση των πινάκων εικόνας
2. εκκινήστε το Bitsocial RPC
3. αποκέντρωση φόρουμ
4. εκκινήστε το Bitsocial Network

Αυτή η ακολουθία έχει νόημα επειδή ένα επίπεδο συντονισμού σε επίπεδο οικοσυστήματος είναι πιο χρήσιμο όταν υπάρχουν ήδη πραγματικές εφαρμογές και πραγματική συμπεριφορά χρήστη στο δίκτυο.
