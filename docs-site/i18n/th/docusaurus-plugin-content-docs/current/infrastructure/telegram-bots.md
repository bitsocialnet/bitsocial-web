# บอทโทรเลข

บอท Bitsocial Telegram ติดตามรายชื่อชุมชนลูกค้าบนเครือข่าย Bitsocial และส่งต่อโพสต์ใหม่ไปยังช่องทาง Telegram โดยอัตโนมัติ แต่ละข้อความที่ส่งต่อจะมีปุ่มอินไลน์ที่ลิงก์กลับไปยังโพสต์ต้นฉบับบน 5chan และ Seeedit

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## บอทที่มีอยู่

| บอท            | สถานะ      | คำอธิบาย                                                        |
| -------------- | ---------- | --------------------------------------------------------------- |
| **ฟีด 5chan**  | ใช้งานอยู่ | ตรวจสอบไดเร็กทอรี 5chan ทั้งหมดและส่งต่อโพสต์ใหม่ไปยัง Telegram |
| **Seedit ฟีด** | วางแผน     | จะให้ฟังก์ชันการทำงานแบบเดียวกันสำหรับชุมชน Seeedit             |

## ตั้งค่า

### ข้อกำหนดเบื้องต้น

- โหนด js
- เส้นด้าย
- โทเค็นบอท Telegram (สร้างหนึ่งรายการผ่าน [บอทพ่อ](https://t.me/BotFather))

### การติดตั้ง

โคลนที่เก็บและติดตั้งการขึ้นต่อกัน:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### การกำหนดค่า

สร้างไฟล์ `.env` ในรูทโปรเจ็กต์ด้วยโทเค็นบอทของคุณ:

```env
BOT_TOKEN=your_telegram_bot_token
```

### วิ่ง

เริ่มบอทหลังจากกำหนดค่าสภาพแวดล้อมของคุณ:

```bash
yarn start
```

## รูปแบบการโพสต์

เมื่อบอทส่งต่อโพสต์ไปที่ Telegram จะมีปุ่มอินไลน์สองปุ่ม:

- **ดูบน 5chan** -- เปิดโพสต์ในเว็บไคลเอ็นต์ 5chan
- **ดูบน Seedit** -- เปิดโพสต์ในเว็บไคลเอ็นต์ Seeedit

ซึ่งช่วยให้สมาชิก Telegram สามารถข้ามไปยังกระทู้สนทนาแบบเต็มได้โดยตรงบนไคลเอนต์ใดก็ได้ที่พวกเขาต้องการ
