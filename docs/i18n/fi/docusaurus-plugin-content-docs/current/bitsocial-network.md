# Bitsocial Network

Bitsocial Network on nimi, jota käytetään yleissuunnitelmassa jaetulle sovellusketjutasolle, joka sijaitsisi Bitsocial-sovellusten alla.

Ajatuksena ei ole korvata peer-to-peer sosiaalista kerrosta. Sen tarkoituksena on tarjota yhteiset taloudelliset ja koordinointialustat, joita sosiaaliset sovellukset tarvitsevat, jos ne aikovat kilpailla vakiintuneiden alustojen kanssa.

## Mitä se tarkoittaa valtaan

- hajautetut Bitsocial-alueet, kuten `.bso`
- jaettu likviditeetti sovellusten välillä
- kestävät kaupallistamiskiskot
- rahoitusrakenteet, joita pankkien tai alustojen on vaikea tukahduttaa
- verkkovaikutuksia, jotka eivät riipu siitä, että yksi yritys omistaa koko pinon

## Miksi sillä on väliä

Yhteisöjen ja identiteettien hajauttaminen on välttämätöntä, mutta se ei riitä koko sosiaalisen median hajauttamiseen.

Jos taloudellinen kerros riippuu edelleen kourallisesta keskitetyistä kiskoista, sovelluksia on edelleen helppo painostaa, poistaa alustasta tai kärsiä taloudellisesti nälkään. Bitsocial Network on ehdotettu vastaus toiseen riippuvuuskerrokseen.

## Suhde muuhun Bitsocialiin

Bitsocial Networkia ehdotetaan jaetuksi infrastruktuuriksi Bitsocial-sovelluksille, ei tuotteeksi, jonka kanssa käyttäjät ovat vuorovaikutuksessa suoraan päivittäin.

Yleisölle tuleva tulos olisi:

- yhteisöt pysyvät toistensa kanssa
- sovellukset pysyvät erilaisina
- kaupallistamisesta ja nimeämisestä tulee kestävämpää
- arvo voi liikkua ekosysteemissä ilman keskitettyä alustan omistajaa

## Miksi tämä kuuluu vaiheeseen 4

Yleissuunnitelma sijoittaa tämän aikaisempien sovellustason todistuspisteiden jälkeen:

1. hajauttaa kuvataulut
2. käynnistä Bitsocial RPC
3. hajauttaa foorumit
4. käynnistää Bitsocial Network

Tämä järjestys on järkevä, koska ekosysteemitason koordinointikerros on hyödyllisempi, kun verkossa on jo todellisia sovelluksia ja todellista käyttäjän käyttäytymistä.
