# i18n سياسة اللغة الافتراضية

يقوم الزائرون المجهولون الآن بتحليل اللغة بهذا الترتيب:

1. معلمة الاستعلام `?lang=`
2. اختيار محدد اللغة المحفوظة من `localStorage`
3. افتراضي مجهول مدرك للمنطقة:
   - فرض اللغة الإنجليزية في المناطق ذات الكثافة السكانية العالية المدرجة صراحةً في اللغة الإنجليزية
   - بخلاف ذلك، استخدم لغة المتصفح/الجهاز عندما تكون إحدى اللغات المدعومة لدينا
4. ارجع إلى `en`

## مصدر الأساس

- [ المؤشر إي آي أف الكفاءة اللغة الإنجليزية 2025](https://www.ef.edu/epi/) لإتقان اللغة الإنجليزية على مستوى الدولة
- [تعداد سنغافورة 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) لسنغافورة، حيث كانت اللغة الإنجليزية هي اللغة المنزلية الأكثر تحدثًا بالنسبة لـ 48.3% من المقيمين الذين تزيد أعمارهم عن 5 سنوات

## المناطق الافتراضية الإنجليزية المشفرة

`AT`، `BE`، `CH`، `CZ`، `DE`، `DK`، `FI`، `GR`، `HU`، `MY`، `NL`، `NO`، `PH`، `PL`، `PT`، `RO`، `SE`، `SG`
