# i18n のデフォルト言語ポリシー

匿名の訪問者は次の順序で言語を解決するようになりました。

1. `?lang=` クエリパラメータ
2. `localStorage` からの言語セレクターの選択を保存しました
3. 地域を意識した匿名のデフォルト:
   - 明示的にリストされている英語の多い地域では英語を強制する
   - それ以外の場合は、サポートされているロケールの 1 つである場合は、ブラウザー/デバイスの言語を使用します。
4. `en` にフォールバックします

## ソースベースライン

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) 国レベルの英語能力
- [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) シンガポールでは、5 歳以上の居住者の 48.3% にとって英語が最も頻繁に話される家庭言語でした。

## エンコードされた英語のデフォルト地域

`AT`, `BE`, `CH`, `CZ`, `DE`, `DK`, `FI`, `GR`, `HU`, `MY`, `NL`, `NO`, `PH`, `PL`, `PT`, `RO`, `SE`, `SG`
