# seedit

Seedit は、Bitsocial プロトコルに基づいて構築された、古いスタイルの Reddit に代わるサーバーレス、管理者不要の完全分散型の代替手段です。これにより、ユーザーは自分のコミュニティに対する完全な所有権を得ることができます。誰でもコミュニティを作成でき、複数のコミュニティがデフォルトのリスト スロットをめぐって競合することができます。

## コミュニティの仕組み

seedit は、次のようなコミュニティにコンテンツを編成します。サブレディット。各コミュニティは作成者によって所有され、集中管理なしで動作します。複数のコミュニティが同じデフォルト スロットをターゲットにしている場合、そのポジションをめぐって競合する可能性があります。

### コミュニティの作成

新しいコミュニティを作成するには、次の 2 つの方法があります。

1. **デスクトップ クライアント** - デスクトップ アプリケーションの組み込み GUI を使用して、コミュニティをセットアップおよび構成します。
2. **bitsocial-cli** — CLI ツールを実行して、コマンド ラインからコミュニティを作成および管理します。

### デフォルト コミュニティ リスト

フロント ページに表示されるデフォルト コミュニティのリストは、リスト リポジトリ内の `default-multisub.json` で定義されます。コミュニティをデフォルトに追加するには、コミュニティの詳細を含むプル リクエストを送信します。
