# 5ちゃん

5chan は、Bitsocial プロトコルで動作するサーバーレス、管理者不要の完全分散型画像掲示板です。これは、分散型所有権を導入しながら、おなじみのイメージボード ディレクトリ構造に従います。誰でもボードを作成でき、投票メカニズムを通じて複数のボードが同じディレクトリ スロットを争うことができます。

## ボードの仕組み

5chan は、古典的なディレクトリ レイアウト (`/b/`、`/g/` など) を使用してコンテンツをボードに整理します。中央管理者がすべてのボードを管理する従来の画像ボードとは異なり、5chan ではあらゆるユーザーが独自のボードを作成し、完全に所有することができます。複数のボードが同じディレクトリ スロットをターゲットにしている場合、投票を通じてそのポジションを競います。

### ボードの作成

新しいボードを作成するには、`bitsocial-cli` をピアツーピア ノードとして実行する必要があります。これにより、中央サーバーに依存せずにボードが分散型でホストされるようになります。

### ディレクトリの割り当て

ディレクトリ スロットの割り当て (どのボードがどのパスに表示されるか) は現在、`5chan-directories.json` ファイルへの GitHub プル リクエストを通じて管理されています。これは一時的なプロセスです。将来のリリースでは、アプリ内ボードの作成と、ディレクトリ割り当てを自動的に処理する pubsub ベースの投票がサポートされる予定です。

## 内部構造

内部では、5chan はネットワーク インタラクションに共有 Bitsocial プロトコル クライアント層を使用します。 5chan.app の Web アプリは、ブラウザー P2P が詳細設定から有効になっている場合、ブラウザーで Helia ノードを実行することもできるため、リーダーは集中型 IPFS ゲートウェイなしでピアからロードできます。ピアツーピア プロトコルに関する注意事項のブラウザ P2P セクションを参照してください。
