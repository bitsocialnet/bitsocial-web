# นโยบายภาษาเริ่มต้นของ i18n

ขณะนี้ผู้เยี่ยมชมที่ไม่ระบุชื่อแก้ไขภาษาตามลำดับนี้:

1. พารามิเตอร์แบบสอบถาม `?lang=`
2. ตัวเลือกตัวเลือกภาษาที่บันทึกไว้จาก `localStorage`
3. ค่าเริ่มต้นที่ไม่ระบุชื่อที่ทราบภูมิภาค:
   - บังคับภาษาอังกฤษในภูมิภาคที่มีภาษาอังกฤษหนาแน่นที่ระบุไว้อย่างชัดเจน
   - มิฉะนั้นให้ใช้ภาษาของเบราว์เซอร์/อุปกรณ์เมื่อเป็นหนึ่งในสถานที่ที่เราสนับสนุน
4. ถอยกลับไปที่ `en`

## ที่มา ข้อมูลพื้นฐาน

- [ดัชนีความสามารถทางภาษาอังกฤษของ EF 2025](https://www.ef.edu/epi/) for country-level English proficiency
- [ และสำมะโนติดตามของสิงคโปร์ พ.ศ. 2563](https://www.singstat.gov.sg/modules/infographics/census-2020) สำหรับสิงคโปร์ โดยที่ภาษาอังกฤษเป็นภาษาหลักที่พูดกันมากที่สุดสำหรับ 48.3% ของผู้พักอาศัยที่มีอายุ 5 ปีขึ้นไป
