# seedit

seedit เป็นทางเลือกแบบไร้เซิร์ฟเวอร์ ไร้ผู้ดูแลระบบ และกระจายอำนาจเต็มรูปแบบ แทน Reddit แบบเก่า ที่สร้างขึ้นบนโปรโตคอล Bitsocial มันช่วยให้ผู้ใช้เป็นเจ้าของชุมชนของตนโดยสมบูรณ์ — ใครๆ ก็สามารถสร้างได้ และหลายชุมชนสามารถแข่งขันกันเพื่อชิงช่องรายการเริ่มต้นได้

## ดาวน์โหลด

| แพลตฟอร์ม  | ลิงค์                                                  |
| ---------- | ------------------------------------------------------ |
| เว็บ       | [seedit.app](https://seedit.app)                       |
| เว็บ (ENS) | `seedit.eth` ผ่านเบราว์เซอร์ Brave หรือ IPFS Companion |
| เดสก์ท็อป  | ใช้งานได้กับ Mac, Windows และ Linux                    |
| มือถือ     | ใช้งานได้กับ Android                                   |

## ชุมชนทำงานอย่างไร

seedit จัดระเบียบเนื้อหาออกเป็นชุมชนที่คล้ายกับ subreddits แต่ละชุมชนเป็นของผู้สร้างและดำเนินงานโดยไม่มีการบริหารแบบรวมศูนย์ เมื่อหลายชุมชนกำหนดเป้าหมายช่องเริ่มต้นเดียวกัน ชุมชนเหล่านั้นก็สามารถแข่งขันเพื่อชิงตำแหน่งนั้นได้

### การสร้างชุมชน

มีสองวิธีในการสร้างชุมชนใหม่:

1. **ไคลเอ็นต์เดสก์ท็อป** — ใช้ GUI ในตัวในแอปพลิเคชันเดสก์ท็อปเพื่อตั้งค่าและกำหนดค่าชุมชนของคุณ
2. **bitsocial-cli** — เรียกใช้เครื่องมือ CLI เพื่อสร้างและจัดการชุมชนจากบรรทัดคำสั่ง

### รายการชุมชนเริ่มต้น

รายการชุมชนเริ่มต้นที่ปรากฏบนหน้าแรกถูกกำหนดไว้ใน `default-multisub.json` ภายในที่เก็บรายการ หากต้องการเพิ่มชุมชนของคุณเป็นค่าเริ่มต้น ให้ส่งคำขอดึงข้อมูลพร้อมรายละเอียดของชุมชนของคุณ
