# Bitsocial Network

Bitsocial Network เป็นชื่อที่ใช้ในแผนหลักสำหรับเลเยอร์ appchain ที่ใช้ร่วมกันซึ่งจะอยู่ภายใต้แอป Bitsocial

แนวคิดนี้ไม่ใช่การแทนที่เลเยอร์ทางสังคมแบบเพียร์ทูเพียร์ มันคือการจัดหาพื้นฐานทางเศรษฐกิจที่ใช้ร่วมกันและการประสานงานที่แอปโซเชียลจำเป็นต้องใช้หากพวกเขาจะแข่งขันกับแพลตฟอร์มที่มีหน้าเดิมในวงกว้าง

## อำนาจหมายถึงอะไร

- โดเมน Bitsocial แบบกระจายอำนาจ เช่น `.bso`
- แบ่งปันสภาพคล่องระหว่างแอพ
- รางการสร้างรายได้ที่ทนทาน
- โครงสร้างทางการเงินที่ยากสำหรับธนาคารหรือแพลตฟอร์มที่จะปิดกั้น
- เอฟเฟกต์เครือข่ายที่ไม่ได้ขึ้นอยู่กับบริษัทเดียวที่เป็นเจ้าของสแต็กทั้งหมด

## ทำไมมันถึงสำคัญ

การกระจายอำนาจให้กับชุมชนและอัตลักษณ์เป็นสิ่งที่จำเป็น แต่ยังไม่เพียงพอในการกระจายอำนาจของโซเชียลมีเดียทั้งหมด

หากชั้นทางเศรษฐกิจยังคงขึ้นอยู่กับรางที่รวมศูนย์จำนวนหนึ่ง แอปต่างๆ ก็ยังคงง่ายต่อการกดดัน เลิกใช้แพลตฟอร์ม หรือขาดแคลนทางการเงิน Bitsocial Network เป็นคำตอบที่นำเสนอสำหรับการพึ่งพาชั้นที่สองนั้น

## ความสัมพันธ์กับส่วนที่เหลือของ Bitsocial

Bitsocial Network ได้รับการเสนอให้เป็นโครงสร้างพื้นฐานที่ใช้ร่วมกันสำหรับแอป Bitsocial ไม่ใช่ผลิตภัณฑ์ที่ผู้ใช้โต้ตอบด้วยโดยตรงทุกวัน

ผลลัพธ์ที่เปิดเผยต่อสาธารณะจะเป็น:

- ชุมชนอยู่แบบเพียร์ทูเพียร์
- แอพยังคงความแตกต่าง
- การสร้างรายได้และการตั้งชื่อมีความคงทนมากขึ้น
- มูลค่าสามารถเคลื่อนข้ามระบบนิเวศได้โดยไม่ต้องสร้างเจ้าของแพลตฟอร์มแบบรวมศูนย์ขึ้นมาใหม่

## เหตุใดจึงอยู่ในระยะที่ 4

3. กระจายอำนาจฟอรัม
4. เปิดตัว Bitsocial Network
