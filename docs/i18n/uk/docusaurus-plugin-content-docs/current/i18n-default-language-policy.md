# Мовна політика i18n за умовчанням

Анонімні відвідувачі тепер розпізнають мову в такому порядку:

1. Параметр запиту `?lang=`
2. Збережено вибір селектора мови з `localStorage`
3. Анонімне значення за умовчанням з урахуванням регіону:
   - форсувати англійську мову в явно перерахованих регіонах з інтенсивним використанням англійської мови
   - інакше використовуйте мову браузера/пристрою, якщо це одна з підтримуваних нами мов
4. Поверніться до `en`

## Вихідна базова лінія

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) для рівня володіння англійською на рівні країни
- [Перепис населення Сінгапуру 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) для Сінгапуру, де англійська була найпоширенішою рідною мовою для 48,3% мешканців віком від 5 років

## Закодовані англійські регіони за замовчуванням

`AT`, `BE`, `CH`, `CZ`, `DE`, `DK`, `FI`, `GR`, `HU`, `MY`, `NL`, `NO`, `PH`, `PL`, `PT`, `RO`, `SE`, `SG`
