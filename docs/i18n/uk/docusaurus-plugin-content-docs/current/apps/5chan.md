# 5chan

5chan — це повністю децентралізована графічна панель без сервера, адміністратора, яка працює на протоколі Bitsocial. Він дотримується знайомої структури каталогів іміджбордів, одночасно впроваджуючи децентралізоване право власності — будь-хто може створити дошку, і кілька дошок можуть конкурувати за один і той самий слот каталогу через механізм голосування.

## Завантаження

| Платформа    | Посилання                         |
| ------------ | --------------------------------- |
| Веб          | [5chan.app](https://5chan.app)    |
| Робочий стіл | Доступно для Mac, Windows і Linux |
| Мобільний    | Доступно для Android              |

## Як працюють дошки

5chan організовує вміст у дошках, використовуючи класичний макет каталогу (наприклад, `/b/`, `/g/`). На відміну від традиційних іміджбордів, де центральний адміністратор контролює кожну дошку, 5chan дозволяє будь-якому користувачеві створювати власну дошку та повністю володіти нею. Коли кілька дощок націлені на одне й те саме місце каталогу, вони змагаються за цю позицію шляхом голосування.

### Створення дошки

Щоб створити нову дошку, вам потрібно запустити `bitsocial-cli` як одноранговий вузол. Це гарантує, що ваша дошка розміщена децентралізовано, не покладаючись на будь-який центральний сервер.

### Присвоєння каталогу

Призначення слотів каталогу (яка дошка з’являється на якому шляху) наразі керується через запити GitHub на отримання файлу `5chan-directories.json`. Це тимчасовий процес — майбутні випуски підтримуватимуть створення дошок у програмі та голосування на основі pubsub для автоматичного призначення каталогів.

## Внутрішні органи

Під капотом 5chan використовує рівень API plebbit-js для взаємодії протоколу. Як зазначалося в попередженні вище, ці внутрішні посилання все ще містять застаріле найменування, яке передувало ребрендингу Bitsocial.
