# i18n డిఫాల్ట్ భాషా విధానం

అనామక సందర్శకులు ఇప్పుడు ఈ క్రమంలో భాషను పరిష్కరిస్తారు:

1. `?lang=` ప్రశ్న పారామ్
2. `localStorage` నుండి భాష ఎంపిక సాధనం సేవ్ చేయబడింది
3. ప్రాంతం-అవగాహన అనామక డిఫాల్ట్:
   - స్పష్టంగా జాబితా చేయబడిన ఆంగ్ల-భారీ ప్రాంతాలలో ఆంగ్లాన్ని బలవంతం చేయండి
   - లేకుంటే బ్రౌజర్/పరికర భాష మా మద్దతు ఉన్న లొకేల్‌లలో ఒకటిగా ఉన్నప్పుడు దాన్ని ఉపయోగించండి
4. `en`కి తిరిగి వెళ్లండి

## మూలాధారం

- దేశ-స్థాయి ఆంగ్ల నైపుణ్యం కోసం [EF English Proficiency Index 2025](https://www.ef.edu/epi/).
- సింగపూర్ కోసం [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020), ఇక్కడ 5+ వయస్సు గల నివాసితులలో 48.3% మందికి ఇంగ్లీషు ఎక్కువగా మాట్లాడే ఇంటి భాష
