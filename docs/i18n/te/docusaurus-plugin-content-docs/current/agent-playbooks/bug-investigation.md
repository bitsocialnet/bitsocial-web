# బగ్ ఇన్వెస్టిగేషన్ వర్క్‌ఫ్లో

నిర్దిష్ట ఫైల్/లైన్/కోడ్ బ్లాక్‌లో బగ్ నివేదించబడినప్పుడు దీన్ని ఉపయోగించండి.

## తప్పనిసరి మొదటి దశ

సవరించడానికి ముందు, సంబంధిత కోడ్ కోసం git చరిత్రను తనిఖీ చేయండి. మునుపటి కంట్రిబ్యూటర్‌లు ఎడ్జ్ కేస్/పర్యావరణ కోసం ప్రవర్తనను ప్రవేశపెట్టి ఉండవచ్చు.

## వర్క్‌ఫ్లో

1. ఫైల్/ఏరియా కోసం ఇటీవలి కమిట్ టైటిల్‌లను (టైటిల్‌లు మాత్రమే) స్కాన్ చేయండి:

```bash
# నిర్దిష్ట ఫైల్ కోసం ఇటీవలి కమిట్ టైటిల్స్
git log --oneline -10 -- src/components/post-desktop/post-desktop.tsx

# నిర్దిష్ట లైన్ పరిధి కోసం ఇటీవలి కమిట్ టైటిల్స్
git blame -L 120,135 src/components/post-desktop/post-desktop.tsx
```

2. స్కోప్డ్ డిఫ్‌లతో సంబంధిత కమిట్‌లను మాత్రమే తనిఖీ చేయండి:

```bash
# ఒక ఫైల్ కోసం కమిట్ మెసేజ్ + తేడాను చూపించు
git show <commit-hash> -- path/to/file.tsx
```

3. పునరుత్పత్తిని కొనసాగించండి మరియు చరిత్ర సందర్భాన్ని అర్థం చేసుకున్న తర్వాత పరిష్కరించండి.

## ట్రబుల్షూటింగ్ నియమం

బ్లాక్ చేయబడినప్పుడు, ఇటీవలి పరిష్కారాలు/పరిష్కారాల కోసం వెబ్‌లో శోధించండి.
