# seedit

seedit היא אלטרנטיבה חסרת שרתים, ניהול ומבוזרת לחלוטין ל-Reddit בסגנון ישן, הבנויה על פרוטוקול Bitsocial. זה נותן למשתמשים בעלות מלאה על הקהילות שלהם - כל אחד יכול ליצור אחת, וקהילות מרובות יכולות להתחרות על משבצות ברירת מחדל.

## הורדות

| פלטפורמה      | קישור                                         |
| ------------- | --------------------------------------------- |
| אינטרנט       | [seedit.app](https://seedit.app)              |
| אינטרנט (ENS) | `seedit.eth` דרך דפדפן אמיץ או IPFS Companion |
| שולחן עבודה   | זמין עבור Mac, Windows ולינוקס                |
| נייד          | זמין עבור אנדרואיד                            |

## כיצד פועלות קהילות

seedit מארגן תוכן בקהילות בדומה ל-subreddits. כל קהילה נמצאת בבעלות היוצר שלה ופועלת ללא ניהול ריכוזי. כאשר קהילות מרובות מכוונות לאותה משבצת ברירת מחדל, הן יכולות להתחרות על המיקום הזה.

### יצירת קהילה

ישנן שתי דרכים ליצור קהילה חדשה:

1. **לקוח שולחני** - השתמש ב-GUI המובנה ביישום שולחן העבודה כדי להגדיר ולהגדיר את הקהילה שלך.
2. **bitsocial-cli** - הפעל את כלי ה-CLI כדי ליצור ולנהל קהילות משורת הפקודה.

### רשימת ברירת מחדל לקהילה

רשימת קהילות ברירת המחדל המופיעות בעמוד הראשון מוגדרת ב-`default-multisub.json` בתוך מאגר הרשימות. כדי להוסיף את הקהילה שלך לברירות המחדל, שלח בקשת משיכה עם פרטי הקהילה שלך.
