# Voucher Challenge

אתגר השובר הוא מנגנון אנטי ספאם שמסגיר את פרסום התוכן מאחורי קודי שוברים ייחודיים. במקום להסתמך על זיהוי אוטומטי, הוא מעביר את האמון לבעל הקהילה, שמפיץ קודים באופן ידני לאנשים שהם סומכים עליהם.

**קוד מקור:** [github.com/bitsocialnet/voucher-challenge](https://github.com/bitsocialnet/voucher-challenge)

## איך זה עובד

1. בעל קהילה יוצר קוד שובר ייחודי אחד או יותר.
2. הבעלים מפיץ את הקודים האלה למחברים מהימנים דרך ערוץ לבחירתם (הודעה ישירה, דואר אלקטרוני, באופן אישי וכו').
3. כאשר מחבר מנסה לפרסם, מערכת האתגר מבקשת ממנו להזין קוד שובר.
4. הקוד מאומת -- אם הוא מקורי ועדיין לא נעשה בו שימוש, הפרסום יתקבל.

כל קוד שובר קשור למחבר ספציפי לאחר מימוש, ומונע שימוש חוזר על ידי אחרים.

## מתי להשתמש בו

אתגר השוברים מתאים ביותר ל:

- **קהילות להזמנה בלבד** שבהן החברות מוגבלת בכוונה.
- **חללים מאוצרים** שבהם הבעלים בודק באופן אישי כל משתתף.
- **סביבות בעלות אמון גבוה** שבהן ניקוד ספאם אוטומטי מיותר או לא רצוי.

מכיוון שהוא מצריך הפצת קוד ידנית, הוא אינו מתרחב לקהילות פתוחות גדולות. עבור תרחישים אלה, שקול [Spam Blocker](./spam-blocker.md) או [EVM Contract Call Challenge](./evm-contract-call.md) במקום זאת.

## אינטגרציה

Voucher Challenge מתחבר לאותו ממשק אתגר המשמש חבילות אנטי-ספאם אחרות במערכת האקולוגית של Bitsocial. בעלי קהילה מאפשרים זאת דרך הגדרות הקהילה שלהם, והאתגר מוצג לכותבים באופן אוטומטי כאשר הם מנסים לפרסם.
