# CLI

:::warning میراثی نام
یہ پیکج فی الحال اس کے اپ اسٹریم انحصار سے وراثت میں ملنے والے نام سازی کنونشنز کا استعمال کرتا ہے۔ کمانڈز، آؤٹ پٹ اور کنفیگریشن میں "plebbit" کے حوالے مستقبل کی ریلیز میں "bitsocial" میں منتقل ہو جائیں گے۔ فعالیت غیر متاثر ہے۔
:::

`bitsocial-cli` بٹسوشل پروٹوکول بیک اینڈ کے ساتھ تعامل کے لیے ایک کمانڈ لائن ٹول ہے۔ یہ آپ کو مقامی P2P ڈیمون چلانے، کمیونٹیز بنانے اور ترتیب دینے، اور مواد شائع کرنے دیتا ہے -- یہ سب کچھ ٹرمینل سے۔

یہ `plebbit-js` کے اوپر بنایا گیا ہے اور اسے [5chan](/apps/5chan/) اور [Seedit](/apps/seedit/) اور کمیونٹی تخلیق کے لیے استعمال کیا جاتا ہے۔ انتظام۔

## تنصیب

پہلے سے تیار کردہ بائنریز ونڈوز، میک او ایس اور لینکس کے لیے دستیاب ہیں۔ اپنے پلیٹ فارم کے لیے تازہ ترین ریلیز GitHub سے ڈاؤن لوڈ کریں:

**[GitHub ریلیز سے ڈاؤن لوڈ کریں](https://github.com/bitsocialnet/bitsocial-cli/releases)**

ڈاؤن لوڈ کرنے کے بعد، بائنری کو قابل عمل بنائیں (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## Demon کو چلانا

CLI کا سب سے عام استعمال بٹسوشل نوڈ چلانا ہے۔ ڈیمون P2P نیٹ ورکنگ پرت کو شروع کرتا ہے اور ایک مقامی API کو بے نقاب کرتا ہے جس سے کلائنٹ جڑ سکتے ہیں۔

```bash
bitsocial-cli daemon
```

پہلے لانچ پر، ڈیمون آپ کے نوڈ، کمیونٹیز اور سیٹنگز کو منظم کرنے کے لیے براؤزر پر مبنی گرافیکل انٹرفیس **WebUI** سے لنک کرتا ہے۔ اگر آپ ٹرمینل کمانڈز پر GUI کو ترجیح دیتے ہیں تو یہ مفید ہے۔

## کلیدی کمانڈز

| کمانڈ               | تفصیل                                                   |
| ------------------- | ------------------------------------------------------- |
| `daemon`            | Bitsocial P2P نوڈ شروع کریں                             |
| `create subplebbit` | ایک نئی کمیونٹی بنائیں                                  |
| `subplebbit edit`   | کمیونٹی کی ترتیبات کو اپ ڈیٹ کریں (عنوان، تفصیل، قواعد) |
| `subplebbit list`   | اس نوڈ پر میزبان کمیونٹیز کی فہرست بنائیں               |
| `subplebbit start`  | ایک مخصوص کمیونٹی کی خدمت شروع کریں                     |
| `subplebbit stop`   | کسی مخصوص کمیونٹی کی خدمت کرنا بند کریں                 |

دستیاب اختیارات اور جھنڈے دیکھنے کے لیے `--help` کے ساتھ کوئی بھی کمانڈ چلائیں:

```bash
bitsocial-cli daemon --help
bitsocial-cli create subplebbit --help
```

## عام ورک فلو

نئی کمیونٹی کی میزبانی کے لیے ایک عام سیٹ اپ فلو:

```bash
# 1. ڈیمون شروع کریں۔
bitsocial-cli daemon

# 2. دوسرے ٹرمینل میں، ایک کمیونٹی بنائیں
bitsocial-cli create subplebbit

# 3. کمیونٹی کو ترتیب دیں۔
bitsocial-cli subplebbit edit <address> --title "My Community" --description "A decentralized forum"

# 4. اسے پیش کرنا شروع کریں۔
bitsocial-cli subplebbit start <address>
```

کمیونٹی اب بٹسوشل نیٹ ورک پر لائیو ہے اور کسی بھی ہم آہنگ کلائنٹ سے قابل رسائی ہے۔
