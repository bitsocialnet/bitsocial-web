# امیج بورڈ کلائنٹ بنائیں

فیز 1 ایک سرکاری ایپ کے بارے میں نہیں ہے جس میں پورے زمرے کا احاطہ کیا گیا ہو۔ 5chan پہلا ثبوت نقطہ ہے، لیکن اصل مقصد ایک وسیع امیج بورڈ ایکو سسٹم ہے: مختلف بصری زبانوں، اعتدال سے متعلق ڈیفالٹس، ڈائریکٹری ماڈلز، اور ٹارگٹ کمیونٹیز کے ساتھ متعدد بٹسوشل کلائنٹس۔

## فیز 1 کی کیا ضرورت ہے۔

- مین اسٹریم آن بورڈنگ کے لیے 4chan طرز کے مانوس کلائنٹس
- مختلف ثقافتوں اور بورڈ بنڈلوں کے ساتھ Altchan سے متاثر کلائنٹس
- موبائل فرسٹ یا کم بینڈوتھ کلائنٹس
- سنگل کمیونٹی یا مخصوص کلائنٹس جن کے ساتھ مضبوط رائے رکھنے والے ڈیفالٹس ہیں۔
- بہتر اعتدال، میڈیا، آن بورڈنگ، یا دریافت کا بہاؤ پہلی ایپ کے ساتھ بھیجنے کے مقابلے میں

## مدد کرنے کا تیز ترین طریقہ

اگر آپ شپنگ کا مختصر ترین راستہ چاہتے ہیں تو پہلے براہ راست 5chan میں تعاون کریں:

- لائیو ایپ کو [5chan.app](https://5chan.app)
- [github.com/bitsocialnet/5chan](https://github.com/bitsocialnet/5chan) پر ماخذ کا جائزہ لیں
- بلڈر چیٹ میں شامل ہوں [t.me/fivechandev](https://t.me/fivechandev)

## اپنا کلائنٹ بنائیں

اگر 5chan کمیونٹی یا انٹرفیس سے مماثل نہیں ہے جو آپ چاہتے ہیں، اس کے بجائے ایک علیحدہ کلائنٹ بنائیں۔ ہم آہنگ Bitsocial کلائنٹس ایک ہی نیٹ ورک کا اشتراک کر سکتے ہیں ایک ہی مصنوعات کے فیصلوں کا اشتراک کیے بغیر۔

1. پروٹوکول کا سامنا کرنے والے ٹولز سیکھیں:
   - [بٹسوشل ری ایکٹ ہکس](../react-hooks/)
   - [بٹسوشل CLI](../cli/)
2. فیصلہ کریں کہ آپ اصل میں کس قسم کا امیج بورڈ بنا رہے ہیں۔
پہلے بورڈ کا ڈھانچہ، شناخت کے مفروضے، اعتدال پسندی کا نمونہ، دریافت کا بہاؤ، اور بصری زبان کا انتخاب کریں۔
3. عمل درآمد کا راستہ منتخب کریں جو پروجیکٹ کے مطابق ہو۔
فورک 5chan اگر آپ ایک مانوس امیج بورڈ بیس کے ساتھ تیزی سے آگے بڑھنا چاہتے ہیں۔ اگر UI یا تعامل کے ماڈل کو یکسر مختلف ہونے کی ضرورت ہو تو نئی شروعات کریں۔
4. ایک تنگ پہلا ورژن بھیجیں۔
ایک کلائنٹ جو ایک حقیقی کمیونٹی کی اچھی طرح سے خدمت کرتا ہے ایک مبہم کلون سے زیادہ قیمتی ہے جس کا مقصد ہر کسی کو مطمئن کرنا ہے۔
5. نتیجہ شائع کریں اور کمیونٹیز کو اس کی جانچ کرنے دیں۔
بٹسوشل اس وقت بہتر ہوتا ہے جب باہر کے بلڈر رائے دہندگان کو بھیجتے ہیں جو سب کچھ کرنے کے لیے ایک سرکاری ایپ کا انتظار کرنے کے بجائے مصنوعات کے معیار پر مقابلہ کرتے ہیں۔

## ڈیزائن کا اصول

بٹسوشل ایک مبارک کلائنٹ رکھنے سے نہیں جیتتا۔ یہ اس وقت جیتتا ہے جب بہت سے کلائنٹس ایک ساتھ رہ سکتے ہیں، کانٹا بنا سکتے ہیں، مہارت حاصل کر سکتے ہیں، اور خدمت کی ضرورتیں پہلی ایپ کو کبھی نہیں ملے گی۔
