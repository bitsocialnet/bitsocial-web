# Telegram Bots

بٹسوشل ٹیلیگرام بوٹس بٹسوشل نیٹ ورک پر کلائنٹ کمیونٹی کی فہرستوں کی نگرانی کرتے ہیں اور خود بخود نئی پوسٹس کو ٹیلیگرام چینلز میں آگے بھیجتے ہیں۔ ہر فارورڈ پیغام میں ان لائن بٹن شامل ہوتے ہیں جو 5chan اور Seedit پر اصل پوسٹ سے واپس لنک کرتے ہیں۔

- **گٹ ہب**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## دستیاب بوٹس

| بوٹ              | حیثیت          | تفصیل                                                                           |
| ---------------- | -------------- | ------------------------------------------------------------------------------- |
| **5چان فیڈ**     | فعال           | تمام 5chan ڈائریکٹریز کی نگرانی کرتا ہے اور نئی پوسٹس کو ٹیلیگرام پر بھیجتا ہے۔ |
| **بیج کی خوراک** | منصوبہ بندی کی | Sedit کمیونٹیز کے لیے وہی فعالیت فراہم کرے گا۔                                  |

## سیٹ اپ

### شرطیں

- Node.js
- سوت
- ٹیلیگرام بوٹ ٹوکن (بذریعہ [BotFather](https://t.me/BotFather))

### تنصیب

ذخیرہ کلون کریں اور انحصار انسٹال کریں:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### کنفیگریشن

اپنے بوٹ ٹوکن کے ساتھ پروجیکٹ روٹ میں `.env` فائل بنائیں:

```env
BOT_TOKEN=your_telegram_bot_token
```

### چل رہا ہے۔

اپنے ماحول کو ترتیب دینے کے بعد بوٹ شروع کریں:

```bash
yarn start
```

## پوسٹ فارمیٹ

جب بوٹ کسی پوسٹ کو ٹیلیگرام پر بھیجتا ہے، تو اس میں دو ان لائن بٹن شامل ہوتے ہیں:

- **5chan پر دیکھیں** -- پوسٹ کو 5chan ویب کلائنٹ میں کھولتا ہے۔
- **سیڈٹ پر دیکھیں** -- پوسٹ کو سیڈٹ ویب کلائنٹ میں کھولتا ہے۔

یہ ٹیلیگرام کے صارفین کو براہ راست مکمل ڈسکشن تھریڈ پر جانے دیتا ہے جس میں وہ کلائنٹ کو ترجیح دیتے ہیں۔
