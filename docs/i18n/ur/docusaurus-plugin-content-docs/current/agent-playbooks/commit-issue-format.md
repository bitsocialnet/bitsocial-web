# کمٹ اور ایشو فارمیٹ

بامعنی کوڈ تبدیلیاں تجویز کرتے یا نافذ کرتے وقت اسے استعمال کریں۔

## تجویز کی شکل کا ارتکاب کریں۔

- **عنوان:** روایتی کمٹ اسٹائل، مختصر، بیک ٹِکس میں لپٹا۔
- کارکردگی کی اصلاح کے لیے `perf` (`fix` نہیں) استعمال کریں۔
- **تفصیل:** اختیاری 2-3 غیر رسمی جملے جو حل کو بیان کرتے ہیں۔ مختصر، تکنیکی، کوئی بلٹ پوائنٹس نہیں۔

مثال:

> **عزم کا عنوان:** `fix: correct date formatting in timezone conversion`
>
> ٹائم زون آفسیٹس کو مناسب طریقے سے سنبھالنے کے لیے `formatDate()` `date-utils.ts` میں اپ ڈیٹ کیا گیا۔

## گٹ ہب ایشو تجویز کی شکل

- **عنوان:** جتنا ممکن ہو مختصر، بیک ٹِکس میں لپٹا۔
- **تفصیل:** 2-3 غیر رسمی جملے جو مسئلہ کو بیان کرتے ہیں (حل نہیں)، گویا ابھی تک حل نہیں ہوا ہے۔

مثال:

> **گٹ ہب مسئلہ:**
>
> - **عنوان:** `Date formatting displays incorrect timezone`
> - **تفصیل:** تبصرے کے ٹائم اسٹیمپ غلط ٹائم زون دکھاتے ہیں جب صارفین مختلف علاقوں سے پوسٹس دیکھتے ہیں۔ `formatDate()` فنکشن صارف کے مقامی ٹائم زون کی ترتیبات کا حساب نہیں رکھتا ہے۔
