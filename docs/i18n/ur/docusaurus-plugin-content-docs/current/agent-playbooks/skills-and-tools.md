# ہنر اور ٹولز

ہنر اور بیرونی ٹولنگ کو ترتیب دیتے وقت اس پلے بک کا استعمال کریں۔

## تجویز کردہ ہنر

### Context7 (لائبریری دستاویزات)

لائبریریوں پر تازہ ترین دستاویزات کے لیے۔

```bash
npx skills add https://github.com/intellectronica/agent-skills --skill context7
```

### پلے رائٹ CLI

براؤزر آٹومیشن کے لیے `playwright-cli` استعمال کریں (نیویگیشن، تعامل، اسکرین شاٹس، ٹیسٹس، ایکسٹرکشن)۔

```bash
npm install -g @playwright/cli@latest
playwright-cli install --skills
```

ہنر انسٹال کرنے کے مقامات:

- `.cursor/skills/playwright-cli/`
- `.claude/skills/playwright-cli/`

### Vercel React بہترین پریکٹسز

ReexNdeep کی کارکردگی کے لیے۔

```bash
npx skills add https://github.com/vercel-labs/agent-skills --skill vercel-react-best-practices
```

### ہنر تلاش کریں۔

کھلے ماحولیاتی نظام سے مہارتیں دریافت/انسٹال کریں۔

```bash
npx skills add https://github.com/vercel-labs/skills --skill find-skills
```

## MCP پالیسی کا استدلال

اس پروجیکٹ کے لیے GitHub MCP اور براؤزر MCP سرورز سے پرہیز کریں کیونکہ وہ اہم ٹول اسکیما/ سیاق و سباق اوور ہیڈ کا اضافہ کرتے ہیں۔

- GitHub آپریشنز: `gh` CLI استعمال کریں۔
- براؤزر آپریشنز: `playwright-cli` استعمال کریں۔
