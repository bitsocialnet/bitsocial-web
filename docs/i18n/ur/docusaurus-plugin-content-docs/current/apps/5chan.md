# 5chan

5chan ایک سرور لیس، ایڈمن لیس، اور مکمل طور پر وکندریقرت امیج بورڈ ہے جو بٹسوشل پروٹوکول پر چلتا ہے۔ یہ وکندریقرت ملکیت کو متعارف کرواتے ہوئے واقف امیج بورڈ ڈائرکٹری ڈھانچے کی پیروی کرتا ہے — کوئی بھی بورڈ بنا سکتا ہے، اور متعدد بورڈ ووٹنگ کے طریقہ کار کے ذریعے ایک ہی ڈائریکٹری سلاٹ کے لیے مقابلہ کر سکتے ہیں۔

## ڈاؤن لوڈز

| پلیٹ فارم | لنک                                    |
| --------- | -------------------------------------- |
| ویب       | [5chan.app](https://5chan.app)         |
| ڈیسک ٹاپ  | میک، ونڈوز اور لینکس کے لیے دستیاب ہے۔ |
| موبائل    | اینڈرائیڈ کے لیے دستیاب ہے۔            |

## بورڈ کیسے کام کرتے ہیں۔

5chan کلاسک ڈائرکٹری لے آؤٹ (جیسے `/b/`، `/g/`) کا استعمال کرتے ہوئے مواد کو بورڈز میں ترتیب دیتا ہے۔ روایتی امیج بورڈز کے برعکس جہاں ایک مرکزی منتظم ہر بورڈ کو کنٹرول کرتا ہے، 5chan کسی بھی صارف کو اپنا بورڈ بنانے اور مکمل طور پر مالک ہونے کی اجازت دیتا ہے۔ جب متعدد بورڈ ایک ہی ڈائرکٹری سلاٹ کو نشانہ بناتے ہیں، تو وہ ووٹنگ کے ذریعے اس پوزیشن کے لیے مقابلہ کرتے ہیں۔

### بورڈ بنانا

نیا بورڈ بنانے کے لیے، آپ کو پیئر ٹو پیئر نوڈ کے طور پر `bitsocial-cli` کو چلانے کی ضرورت ہے۔ یہ یقینی بناتا ہے کہ آپ کا بورڈ کسی مرکزی سرور پر بھروسہ کیے بغیر وکندریقرت انداز میں ہوسٹ کیا گیا ہے۔

### ڈائرکٹری اسائنمنٹس

ڈائرکٹری سلاٹ اسائنمنٹس (کون سا بورڈ کس راستے پر ظاہر ہوتا ہے) فی الحال گٹ ہب کے ذریعے `5chan-directories.json` فائل پر پل ریکوئسٹ کا انتظام کیا جاتا ہے۔ یہ ایک عارضی عمل ہے — مستقبل کی ریلیزز ڈائرکٹری اسائنمنٹس کو خود بخود ہینڈل کرنے کے لیے ایپ بورڈ کی تخلیق اور پبس پر مبنی ووٹنگ کو سپورٹ کریں گی۔

## اندرونی

ہڈ کے نیچے، 5chan اپنے پروٹوکول تعاملات کے لیے plebbit-js API پرت کا استعمال کرتا ہے۔ جیسا کہ اوپر وارننگ میں بتایا گیا ہے، یہ داخلی حوالہ جات اب بھی میراثی نام رکھتے ہیں جو Bitsocial rebrand سے پہلے ہے۔
