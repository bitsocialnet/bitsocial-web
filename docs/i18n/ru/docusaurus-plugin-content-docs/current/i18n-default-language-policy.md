# i18n Языковая политика по умолчанию

Анонимные посетители теперь определяют язык в следующем порядке:

1. Параметр запроса `?lang=`
2. Сохранен выбор языка из `localStorage`.
3. Анонимное значение по умолчанию с учетом региона:
   - принудительно использовать английский язык в явно перечисленных регионах с большим количеством английского языка
   - в противном случае используйте язык браузера/устройства, если он является одним из поддерживаемых нами языков.
4. Вернитесь к `en`.

## Исходный базовый уровень

- [Индекс владения английским языком EF 2025](https://www.ef.edu/epi/) для уровня владения английским языком на уровне страны
- [Перепись населения Сингапура 2020 года](https://www.singstat.gov.sg/modules/infographics/census-2020) для Сингапура, где английский был наиболее часто используемым домашним языком для 48,3% жителей в возрасте 5+

## Закодированные английские регионы по умолчанию

`AT`, `BE`, `CH`, `CZ`, `DE`, `DK`, `FI`, `GR`, `HU`, `MY`, `NL`, `NO`, `PH`, `PL`, `PT`, `RO`, `SE`, `SG`
