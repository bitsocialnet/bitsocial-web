# Voucher Challenge

Voucher Challenge — это механизм защиты от спама, который защищает публикацию контента с помощью уникальных кодов ваучеров. Вместо того, чтобы полагаться на автоматическое обнаружение, он передает доверие владельцу сообщества, который вручную распространяет коды людям, которым они доверяют.

**Исходный код:** [github.com/bitsocialnet/voucher-challenge](https://github.com/bitsocialnet/voucher-challenge)

## Как это работает

1. Владелец сообщества генерирует один или несколько уникальных кодов ваучеров.
2. Владелец распространяет эти коды доверенным авторам по выбранному ими каналу (прямое сообщение, электронная почта, лично и т. д.).
3. Когда автор пытается опубликовать, система конкурса запрашивает код ваучера.
4. Код проверен: если он подлинный и еще не использовался, публикация принимается.

Каждый код ваучера привязывается к конкретному автору после погашения, что предотвращает повторное использование другими лицами.

## Когда его использовать

Voucher Challenge лучше всего подходит для:

- **Сообщества, доступные только по приглашению**, членство в которых намеренно ограничено.
- **Кураторские помещения**, где владелец лично проверяет каждого участника.
- **Среды с высоким уровнем доверия**, в которых автоматическая оценка спама не требуется или нежелательна.

Поскольку он требует распространения кода вручную, он не масштабируется для больших открытых сообществ. Для этих сценариев вместо этого рассмотрите [Spam Blocker](./spam-blocker.md) или [EVM Contract Call Challenge](./evm-contract-call.md).

## Интеграция

Voucher Challenge подключается к тому же интерфейсу, который используется другими пакетами защиты от спама в экосистеме Bitsocial. Владельцы сообществ включают эту функцию в настройках своего сообщества, и задача автоматически предоставляется авторам при попытке опубликовать сообщение.
