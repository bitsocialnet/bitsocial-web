# কমিট এবং ইস্যু ফরম্যাট

অর্থপূর্ণ কোড পরিবর্তনের প্রস্তাব বা বাস্তবায়ন করার সময় এটি ব্যবহার করুন।

## কমিট সাজেশন ফরম্যাট

- **শিরোনাম:** প্রচলিত প্রতিশ্রুতি শৈলী, সংক্ষিপ্ত, ব্যাকটিক্সে মোড়ানো।
- কর্মক্ষমতা অপ্টিমাইজেশনের জন্য `perf` (`fix` নয়) ব্যবহার করুন৷
- **বর্ণনা:** ঐচ্ছিক 2-3টি অনানুষ্ঠানিক বাক্য যা সমাধানের বর্ণনা দেয়। সংক্ষিপ্ত, প্রযুক্তিগত, কোনো বুলেট পয়েন্ট নেই।

উদাহরণ:

> **কমিট শিরোনাম:** `fix: correct date formatting in timezone conversion`
>
> টাইমজোন অফসেটগুলি সঠিকভাবে পরিচালনা করতে `date-utils.ts`-এ `formatDate()` আপডেট করা হয়েছে৷

## গিটহাব ইস্যু সাজেশন ফরম্যাট

- **শিরোনাম:** যতটা সম্ভব ছোট, ব্যাকটিক্সে মোড়ানো।
- **বিবরণ:** 2-3টি অনানুষ্ঠানিক বাক্য যা সমস্যাটি বর্ণনা করে (সমাধান নয়), যেন এখনও অমীমাংসিত।

উদাহরণ:

> **গিটহাব সমস্যা:**
>
> - **শিরোনাম:** `Date formatting displays incorrect timezone`
> - **বিবরণ:** যখন ব্যবহারকারীরা বিভিন্ন অঞ্চলের পোস্টগুলি দেখেন তখন মন্তব্য টাইমস্ট্যাম্পগুলি ভুল টাইমজোন দেখায়৷ `formatDate()` ফাংশন ব্যবহারকারীর স্থানীয় সময় অঞ্চল সেটিংসের জন্য হিসাব করে না।
