# i18n ডিফল্ট ভাষা নীতি

বেনামী দর্শকরা এখন এই ক্রমে ভাষা সমাধান করে:

1. `?lang=` কোয়েরি পরম
2. `localStorage` থেকে ভাষা নির্বাচক পছন্দ সংরক্ষিত
3. অঞ্চল-সচেতন বেনামী ডিফল্ট:
   - স্পষ্টভাবে তালিকাভুক্ত ইংরেজি-ভারী অঞ্চলে ইংরেজি বলুন
   - অন্যথায় ব্রাউজার/ডিভাইস ভাষা ব্যবহার করুন যখন এটি আমাদের সমর্থিত লোকেলগুলির মধ্যে একটি
4. `en` এ ফিরে যান

## উৎস বেসলাইন

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) দেশ-স্তরের ইংরেজি দক্ষতার জন্য
- [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) সিঙ্গাপুরের জন্য, যেখানে 5+ বছর বয়সী 48.3% বাসিন্দাদের জন্য ইংরেজি ছিল সর্বাধিক কথ্য ভাষা
