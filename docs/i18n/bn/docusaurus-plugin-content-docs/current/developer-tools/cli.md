# CLI

:::warning উত্তরাধিকার নামকরণ
এই প্যাকেজটি বর্তমানে এর আপস্ট্রিম নির্ভরতা থেকে উত্তরাধিকারসূত্রে প্রাপ্ত লিগ্যাসি নামকরণের নিয়মগুলি ব্যবহার করে। কমান্ড, আউটপুট এবং কনফিগারেশনে "plebbit" এর উল্লেখগুলি ভবিষ্যতের রিলিজে "bitsocial"-এ স্থানান্তরিত হবে। কার্যকারিতা প্রভাবিত হয় না।
:::

`bitsocial-cli` বিটসোশ্যাল প্রোটোকল ব্যাকএন্ডের সাথে ইন্টারঅ্যাক্ট করার জন্য একটি কমান্ড-লাইন টুল। এটি আপনাকে একটি স্থানীয় P2P ডেমন চালাতে, সম্প্রদায়গুলি তৈরি এবং কনফিগার করতে এবং সামগ্রী প্রকাশ করতে দেয় -- সব টার্মিনাল থেকে।

এটি `plebbit-js` এর উপরে নির্মিত এবং [5chan](/apps/5chan/) এবং [Seedit](/apps/seedit/) সম্প্রদায় তৈরি এবং নোড পরিচালনার জন্য ব্যবহার করে।

## ইনস্টলেশন

পূর্ব-নির্মিত বাইনারিগুলি Windows, macOS এবং Linux-এর জন্য উপলব্ধ। GitHub থেকে আপনার প্ল্যাটফর্মের জন্য সর্বশেষ রিলিজ ডাউনলোড করুন:

**[GitHub রিলিজ থেকে ডাউনলোড করুন](https://github.com/bitsocialnet/bitsocial-cli/releases)**

ডাউনলোড করার পরে, বাইনারি এক্সিকিউটেবল করুন (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## ডেমন চালাচ্ছে

CLI-এর সবচেয়ে সাধারণ ব্যবহার হল বিটসোশ্যাল নোড চালানো। ডেমন P2P নেটওয়ার্কিং স্তর শুরু করে এবং একটি স্থানীয় API প্রকাশ করে যা ক্লায়েন্টরা সংযোগ করতে পারে।

```bash
bitsocial-cli daemon
```

প্রথম লঞ্চে, ডেমন আউটপুট **WebUI**-এর সাথে লিঙ্ক করে, আপনার নোড, সম্প্রদায় এবং সেটিংস পরিচালনার জন্য একটি ব্রাউজার-ভিত্তিক গ্রাফিকাল ইন্টারফেস। আপনি যদি টার্মিনাল কমান্ডের তুলনায় একটি GUI পছন্দ করেন তাহলে এটি কার্যকর।

## কী কমান্ড

| আদেশ                | বর্ণনা                                               |
| ------------------- | ---------------------------------------------------- |
| `daemon`            | Bitsocial P2P নোড শুরু করুন                          |
| `create subplebbit` | একটি নতুন সম্প্রদায় তৈরি করুন                       |
| `subplebbit edit`   | সম্প্রদায় সেটিংস আপডেট করুন (শিরোনাম, বিবরণ, নিয়ম) |
| `subplebbit list`   | এই নোডে হোস্ট করা সম্প্রদায়ের তালিকা করুন           |
| `subplebbit start`  | একটি নির্দিষ্ট সম্প্রদায়ের সেবা করা শুরু করুন       |
| `subplebbit stop`   | একটি নির্দিষ্ট সম্প্রদায়ের সেবা করা বন্ধ করুন       |

উপলব্ধ বিকল্প এবং পতাকাগুলি দেখতে `--help` দিয়ে যেকোনো কমান্ড চালান:

```bash
bitsocial-cli daemon --help
bitsocial-cli create subplebbit --help
```

## সাধারণ ওয়ার্কফ্লো

একটি নতুন সম্প্রদায় হোস্ট করার জন্য একটি সাধারণ সেটআপ প্রবাহ:

```bash
# 1. Start the daemon
bitsocial-cli daemon

# 2. In another terminal, create a community
bitsocial-cli create subplebbit

# 3. Configure the community
bitsocial-cli subplebbit edit <address> --title "My Community" --description "A decentralized forum"

# 4. Start serving it
bitsocial-cli subplebbit start <address>
```

এখন লাইভ নেটওয়ার্ক থেকে নেটওয়ার্ক অ্যাক্সেস করা যায়। কোনো সামঞ্জস্যপূর্ণ ক্লায়েন্ট।
