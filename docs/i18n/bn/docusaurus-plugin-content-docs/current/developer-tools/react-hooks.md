# React Hooks

:::warning উত্তরাধিকার নামকরণ
এই প্যাকেজটি বর্তমানে এর আপস্ট্রিম ফর্ক থেকে উত্তরাধিকারসূত্রে প্রাপ্ত লিগ্যাসি নামকরণ পদ্ধতি ব্যবহার করে। কোড, API এবং কনফিগারেশনে "plebbit"-এর রেফারেন্সগুলি ভবিষ্যতের রিলিজে "bitsocial"-এ স্থানান্তরিত হবে। কার্যকারিতা প্রভাবিত হয় না।
:::

`bitsocial-react-hooks` প্যাকেজ বিটসোশ্যাল প্রোটোকলের সাথে ইন্টারঅ্যাক্ট করার জন্য একটি পরিচিত রিঅ্যাক্ট হুক API প্রদান করে। এটি একটি কেন্দ্রীয় সার্ভারের উপর নির্ভর না করেই ফিড, মন্তব্য এবং লেখকের প্রোফাইল, অ্যাকাউন্ট পরিচালনা, বিষয়বস্তু প্রকাশ এবং সম্প্রদায়গুলিতে সদস্যতা নেওয়া পরিচালনা করে।

এই লাইব্রেরিটি [5chan](/apps/5chan/) এবং অন্যান্য বিটসোশ্যাল ক্লায়েন্ট অ্যাপ্লিকেশন দ্বারা ব্যবহৃত প্রাথমিক ইন্টারফেস।

:::note
`bitsocial-react-hooks` হল `plebbit/plebbit-react-hooks` এর একটি অস্থায়ী কাঁটা যা এআই-সহায়তা উন্নয়নের জন্য রক্ষিত। এটি এনপিএম-এ প্রকাশিত না হয়ে সরাসরি গিটহাব থেকে গ্রহণ করা হয়।
:::

## ইনস্টলেশন

কারণ প্যাকেজটি এখনও npm-এ নেই, এটি সরাসরি GitHub থেকে ইনস্টল করুন, একটি নির্দিষ্ট কমিট হ্যাশে পিন করুন:

```bash
yarn add https://github.com/bitsocialnet/bitsocial-react-hooks.git#<commit-hash>
```

আপনি যে প্রতিশ্রুতি লক্ষ্য করতে চান তার সাথে `<commit-hash>` প্রতিস্থাপন করুন।

## API ওভারভিউ

হুকগুলি কার্যকরী বিভাগে সংগঠিত হয়। নীচে প্রতিটি বিভাগে সর্বাধিক ব্যবহৃত হুকগুলির একটি সারাংশ রয়েছে৷ সম্পূর্ণ স্বাক্ষর, পরামিতি, এবং রিটার্ন প্রকারের জন্য, [GitHub এ সম্পূর্ণ API রেফারেন্স](https://github.com/bitsocialnet/bitsocial-react-hooks) দেখুন।

### অ্যাকাউন্ট

স্থানীয় ব্যবহারকারীর অ্যাকাউন্ট, পরিচয়, এবং সেটিংস পরিচালনা করে।

- `useAccount(accountName?)` -- সক্রিয় (বা নাম দেওয়া) অ্যাকাউন্ট অবজেক্ট ফেরত দেয়
- `useAccounts()` -- স্থানীয়ভাবে সঞ্চিত সমস্ত অ্যাকাউন্ট ফেরত দেয়
- `useAccountComments(options?)` -- সক্রিয় দ্বারা প্রকাশিত মন্তব্য ফেরত দেয় অ্যাকাউন্ট

### মন্তব্য

আনয়ন এবং পৃথক মন্তব্য এবং থ্রেডগুলির সাথে ইন্টারঅ্যাক্ট করে৷

- `useComment(commentCid?)` -- এটির CID দ্বারা একটি একক মন্তব্য নিয়ে আসে
- `useComments(commentCids?)` -- ব্যাচে একাধিক মন্তব্য নিয়ে আসে
- `useEditedComment(comment?)` -- একটি মন্তব্যের সর্বশেষ সম্পাদিত সংস্করণ প্রদান করে

### সম্প্রদায়গুলি

সম্প্রদায়ের মেটাডেটা এবং সেটিংস পুনরুদ্ধার করুন।

- `useSubplebbit(subplebbitAddress?)` -- ঠিকানা দ্বারা একটি সম্প্রদায় নিয়ে আসে
- `useSubplebbits(subplebbitAddresses?)` -- একাধিক সম্প্রদায় নিয়ে আসে
- `useSubplebbitStats(subplebbitAddress?)` -- গ্রাহক এবং পোস্টের সংখ্যা ফেরত দেয়

### লেখক

লেখকের প্রোফাইল এবং মেটাডেটা দেখুন।

- `useAuthor(authorAddress?)` -- একজন লেখকের প্রোফাইল নিয়ে আসে
- `useAuthorComments(options?)` -- একটি নির্দিষ্ট লেখকের মন্তব্য ফেরত দেয়
- `useResolvedAuthorAddress(authorAddress?)` -- একটি মানব-পাঠযোগ্য ঠিকানা (যেমন, ENS) এর প্রোটোকল ঠিকানায় সমাধান করে

### ফিড

সাবস্ক্রাইব করুন এবং কন্টেন্ট ফিড পেজিনেট করুন।

- `useFeed(options?)` -- এক বা একাধিক সম্প্রদায়ের পোস্টের একটি পাতাযুক্ত ফিড ফেরত দেয়
- `useBufferedFeeds(feedOptions?)` -- দ্রুত রেন্ডারিংয়ের জন্য একাধিক ফিড প্রি-বাফার করে
- `useAuthorFeed(authorAddress?)` -- একটি নির্দিষ্ট লেখকের পোস্টের একটি ফিড ফেরত দেয়

### Actions

সামগ্রী প্রকাশ করে এবং লেখার ক্রিয়াকলাপ সম্পাদন করে।

- `usePublishComment(options?)` -- একটি নতুন মন্তব্য বা উত্তর প্রকাশ করুন
- `usePublishVote(options?)` -- একটি আপভোট বা ডাউনভোট করুন
- `useSubscribe(options?)` -- সাবস্ক্রাইব করুন বা সাবস্ক্রাইব করুন সম্প্রদায়

### রাষ্ট্র এবং RPC

সংযোগের অবস্থা পর্যবেক্ষণ করুন এবং একটি দূরবর্তী বিটসোশ্যাল ডেমনের সাথে যোগাযোগ করুন।

- `useClientsStates(options?)` -- IPFS/pubsub ক্লায়েন্টদের সংযোগের অবস্থা ফেরত দেয়
- `usePlebbitRpcSettings()` -- বর্তমান RPC ডেমন কনফিগারেশন ফেরত দেয়
