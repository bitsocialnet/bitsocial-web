# Telegram Bots

বিটসোশ্যাল টেলিগ্রাম বটগুলি বিটসোশ্যাল নেটওয়ার্কে ক্লায়েন্ট সম্প্রদায়ের তালিকা নিরীক্ষণ করে এবং স্বয়ংক্রিয়ভাবে নতুন পোস্টগুলি টেলিগ্রাম চ্যানেলগুলিতে ফরওয়ার্ড করে। প্রতিটি ফরোয়ার্ড করা বার্তায় ইনলাইন বোতাম থাকে যা 5chan এবং Seedit-এর মূল পোস্টের সাথে লিঙ্ক করে।

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## উপলব্ধ বট

| বট             | স্ট্যাটাস | বর্ণনা                                                                      |
| -------------- | --------- | --------------------------------------------------------------------------- |
| **5চ্যান ফিড** | সক্রিয়   | সমস্ত 5chan ডিরেক্টরি নিরীক্ষণ করে এবং টেলিগ্রামে নতুন পোস্ট ফরোয়ার্ড করে। |
| **বীজ ফিড**    | পরিকল্পিত | Seedit সম্প্রদায়ের জন্য একই কার্যকারিতা প্রদান করবে।                       |

## সেটআপ

### পূর্বশর্ত

- Node.js
- সুতা
- একটি টেলিগ্রাম বট টোকেন (একটি তৈরি করুন [BotFather](https://t.me/BotFather)) এর মাধ্যমে)

### ইনস্টলেশন

সংগ্রহস্থল ক্লোন করুন এবং নির্ভরতা ইনস্টল করুন:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### কনফিগারেশন

আপনার বট টোকেন দিয়ে প্রকল্প রুটে একটি `.env` ফাইল তৈরি করুন:

```env
BOT_TOKEN=your_telegram_bot_token
```

### চলছে

আপনার পরিবেশ কনফিগার করার পরে বটটি শুরু করুন:

```bash
yarn start
```

## পোস্ট ফরম্যাট

যখন বট টেলিগ্রামে একটি পোস্ট ফরোয়ার্ড করে, তখন এতে দুটি ইনলাইন বোতাম অন্তর্ভুক্ত থাকে:

- **5chan-এ দেখুন** -- 5chan ওয়েব ক্লায়েন্টে পোস্টটি খোলে।
- ** Seedit এ দেখুন** -- Seedit ওয়েব ক্লায়েন্টে পোস্টটি খোলে।

এটি টেলিগ্রাম গ্রাহকদের তাদের পছন্দের ক্লায়েন্টের সম্পূর্ণ আলোচনার থ্রেডে সরাসরি যেতে দেয়।
