# seedit

seedit হল একটি সার্ভারহীন, প্রশাসকহীন, এবং পুরানো-স্টাইলের Reddit-এর সম্পূর্ণ বিকেন্দ্রীকৃত বিকল্প, যা Bitsocial প্রোটোকলের উপর নির্মিত। এটি ব্যবহারকারীদের তাদের সম্প্রদায়ের উপর সম্পূর্ণ মালিকানা দেয় — যে কেউ একটি তৈরি করতে পারে এবং একাধিক সম্প্রদায় ডিফল্ট তালিকা স্লটের জন্য প্রতিযোগিতা করতে পারে।

## সম্প্রদায়গুলি কীভাবে কাজ করে

seedit সাবরেডিটের মতো সম্প্রদায়গুলিতে বিষয়বস্তু সংগঠিত করে। প্রতিটি সম্প্রদায় তার সৃষ্টিকর্তার মালিকানাধীন এবং কেন্দ্রীভূত প্রশাসন ছাড়াই কাজ করে। যখন একাধিক সম্প্রদায় একই ডিফল্ট স্লট লক্ষ্য করে, তারা সেই অবস্থানের জন্য প্রতিদ্বন্দ্বিতা করতে পারে।

### একটি সম্প্রদায় তৈরি করা

একটি নতুন সম্প্রদায় তৈরি করার দুটি উপায় রয়েছে:

1. **ডেস্কটপ ক্লায়েন্ট** — আপনার সম্প্রদায় সেট আপ এবং কনফিগার করতে ডেস্কটপ অ্যাপ্লিকেশনে অন্তর্নির্মিত GUI ব্যবহার করুন।
2. **bitsocial-cli** — কমান্ড লাইন থেকে সম্প্রদায় তৈরি এবং পরিচালনা করতে CLI টুলটি চালান।

### ডিফল্ট সম্প্রদায় তালিকা

প্রথম পৃষ্ঠায় প্রদর্শিত ডিফল্ট সম্প্রদায়ের তালিকা তালিকা সংগ্রহস্থলের মধ্যে `default-multisub.json` এ সংজ্ঞায়িত করা হয়েছে। আপনার সম্প্রদায়কে ডিফল্টে যুক্ত করতে, আপনার সম্প্রদায়ের বিবরণ সহ একটি পুল অনুরোধ জমা দিন৷
