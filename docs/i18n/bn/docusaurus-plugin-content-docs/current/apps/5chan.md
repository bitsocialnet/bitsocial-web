# 5chan

5chan হল একটি সার্ভারহীন, প্রশাসনবিহীন, এবং সম্পূর্ণ বিকেন্দ্রীভূত ইমেজবোর্ড যা বিটসোশ্যাল প্রোটোকলের উপর চলে৷ এটি বিকেন্দ্রীভূত মালিকানা প্রবর্তন করার সময় পরিচিত ইমেজবোর্ড ডিরেক্টরি কাঠামো অনুসরণ করে — যে কেউ একটি বোর্ড তৈরি করতে পারে, এবং একাধিক বোর্ড একই ডিরেক্টরি স্লটের জন্য একটি ভোটিং প্রক্রিয়ার মাধ্যমে প্রতিদ্বন্দ্বিতা করতে পারে৷

## বোর্ডগুলি কীভাবে কাজ করে

5চান একটি ক্লাসিক বোর্ড ব্যবহার করে বিষয়বস্তু সংগঠিত করে। `/b/`, `/g/`)। প্রথাগত ইমেজবোর্ডের বিপরীতে যেখানে একজন কেন্দ্রীয় প্রশাসক প্রতিটি বোর্ড নিয়ন্ত্রণ করেন, 5chan যেকোনো ব্যবহারকারীকে তাদের নিজস্ব বোর্ড তৈরি করতে এবং সম্পূর্ণ মালিকানার অনুমতি দেয়। যখন একাধিক বোর্ড একই ডিরেক্টরি স্লটকে লক্ষ্য করে, তারা ভোটের মাধ্যমে সেই অবস্থানের জন্য প্রতিদ্বন্দ্বিতা করে।

### একটি বোর্ড তৈরি করা

একটি নতুন বোর্ড তৈরি করতে, আপনাকে পিয়ার-টু-পিয়ার নোড হিসাবে `bitsocial-cli` চালাতে হবে। এটি নিশ্চিত করে যে আপনার বোর্ড কোনো কেন্দ্রীয় সার্ভারের উপর নির্ভর না করে বিকেন্দ্রীভূত পদ্ধতিতে হোস্ট করা হয়েছে।

### ডিরেক্টরি অ্যাসাইনমেন্ট

ডিরেক্টরি স্লট অ্যাসাইনমেন্ট (কোন বোর্ড কোন পথে প্রদর্শিত হয়) বর্তমানে `5chan-directories.json` ফাইলে GitHub পুল অনুরোধের মাধ্যমে পরিচালিত হয়। এটি একটি অস্থায়ী প্রক্রিয়া — ভবিষ্যতের রিলিজগুলি স্বয়ংক্রিয়ভাবে ডিরেক্টরি অ্যাসাইনমেন্টগুলি পরিচালনা করতে অ্যাপ-মধ্যস্থ বোর্ড তৈরি এবং পাবসাব-ভিত্তিক ভোটিং সমর্থন করবে।

## অভ্যন্তরীণ

হুডের নিচে, 5chan তার প্রোটোকল ইন্টারঅ্যাকশনের জন্য plebbit-js API স্তর ব্যবহার করে। উপরের সতর্কতায় যেমন উল্লেখ করা হয়েছে, এই অভ্যন্তরীণ রেফারেন্সগুলি এখনও লিগ্যাসি নামকরণ বহন করে যা বিটসোশ্যাল রিব্র্যান্ডের পূর্ববর্তী।
