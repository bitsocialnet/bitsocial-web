# i18n डिफ़ॉल्ट भाषा नीति

अनाम विज़िटर अब इस क्रम में भाषा का समाधान करते हैं:

1. `?lang=` क्वेरी पैरामीटर
2. `localStorage` से भाषा चयनकर्ता विकल्प सहेजा गया
3. क्षेत्र-जागरूक अनाम डिफ़ॉल्ट:
   - स्पष्ट रूप से सूचीबद्ध अंग्रेजी-भारी क्षेत्रों में अंग्रेजी को मजबूर करें
   - अन्यथा ब्राउज़र/डिवाइस भाषा का उपयोग तब करें जब वह हमारे समर्थित स्थानों में से एक हो
4. `en` पर वापस जाएँ

## स्रोत बेसलाइन

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) देश-स्तरीय अंग्रेजी दक्षता के लिए
- [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) सिंगापुर के लिए, जहां 5+ आयु वर्ग के 48.3% निवासियों के लिए अंग्रेजी सबसे अधिक बोली जाने वाली घरेलू भाषा थी।
