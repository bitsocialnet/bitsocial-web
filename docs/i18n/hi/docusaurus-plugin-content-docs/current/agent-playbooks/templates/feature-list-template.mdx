# फ़ीचर सूची टेम्पलेट

यह पृष्ठ `docs/agent-playbooks/templates/feature-list.template.json` में भेजे गए JSON टेम्पलेट को प्रतिबिंबित करता है ताकि इसे सीधे डॉक्स साइट के अंदर भी ब्राउज़ किया जा सके।

```json
{
  "task": "replace-with-task-slug",
  "last_updated": "YYYY-MM-DD",
  "items": [
    {
      "id": "F001",
      "priority": 1,
      "status": "pending",
      "description": "Describe one end-to-end feature or one reviewable task slice.",
      "verification": ["List the command or user-visible check that proves this item works."],
      "files": [],
      "notes": ""
    }
  ]
}
```
