# CLI

:::warning विरासत नामकरण
यह पैकेज वर्तमान में अपनी अपस्ट्रीम निर्भरता से विरासत में मिली विरासत नामकरण परंपराओं का उपयोग करता है। कमांड, आउटपुट और कॉन्फ़िगरेशन में "प्लेबिट" के संदर्भों को भविष्य के रिलीज में "बिटसोशल" में स्थानांतरित कर दिया जाएगा। कार्यक्षमता अप्रभावित है.
:::

`bitsocial-cli` बिटसोशल प्रोटोकॉल बैकएंड के साथ इंटरैक्ट करने के लिए एक कमांड-लाइन टूल है। यह आपको स्थानीय पी2पी डेमॉन चलाने, समुदायों को बनाने और कॉन्फ़िगर करने और सामग्री प्रकाशित करने की सुविधा देता है - यह सब टर्मिनल से।

इसे `plebbit-js` के शीर्ष पर बनाया गया है और इसका उपयोग समुदाय निर्माण और नोड प्रबंधन के लिए [5chan](/apps/5chan/) और [Seedit](/apps/seedit/) द्वारा किया जाता है।

## स्थापना

पूर्व-निर्मित बायनेरिज़ Windows, macOS और Linux के लिए उपलब्ध हैं। Download the latest release for your platform from GitHub:

**[GitHub रिलीज़ से डाउनलोड करें](https://github.com/bitsocialnet/bitsocial-cli/releases)**

डाउनलोड करने के बाद, बाइनरी को निष्पादन योग्य बनाएं (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## डेमॉन चलाना

सीएलआई का सबसे आम उपयोग बिटसोशल नोड चलाना है। डेमॉन पी2पी नेटवर्किंग परत शुरू करता है और एक स्थानीय एपीआई को उजागर करता है जिससे ग्राहक जुड़ सकते हैं।

```bash
bitsocial-cli daemon
```

पहले लॉन्च पर, डेमॉन **वेबयूआई** से लिंक आउटपुट करता है, जो आपके नोड, समुदायों और सेटिंग्स को प्रबंधित करने के लिए एक ब्राउज़र-आधारित ग्राफिकल इंटरफ़ेस है। यदि आप टर्मिनल कमांड पर GUI पसंद करते हैं तो यह उपयोगी है।

## मुख्य आदेश

| आदेश                | विवरण                                             |
| ------------------- | ------------------------------------------------- |
| `daemon`            | बिटसोशल पी2पी नोड प्रारंभ करें                    |
| `create subplebbit` | एक नया समुदाय बनाएं                               |
| `subplebbit edit`   | सामुदायिक सेटिंग अपडेट करें (शीर्षक, विवरण, नियम) |
| `subplebbit list`   | इस नोड पर होस्ट किए गए समुदायों की सूची बनाएं     |
| `subplebbit start`  | एक विशिष्ट समुदाय की सेवा शुरू करें               |
| `subplebbit stop`   | एक विशिष्ट समुदाय की सेवा करना बंद करें           |

उपलब्ध विकल्प और झंडे देखने के लिए `--help` के साथ कोई भी कमांड चलाएँ:

```bash
bitsocial-cli daemon --help
bitsocial-cli create subplebbit --help
```

## विशिष्ट कार्यप्रवाह

नए समुदाय की मेजबानी के लिए एक सामान्य सेटअप प्रवाह:

```bash
# 1. डेमॉन प्रारंभ करें
bitsocial-cli daemon

# 2. दूसरे टर्मिनल में, एक समुदाय बनाएं
bitsocial-cli create subplebbit

# 3. समुदाय को कॉन्फ़िगर करें
bitsocial-cli subplebbit edit <address> --title "My Community" --description "A decentralized forum"

# 4. इसे परोसना शुरू करें
bitsocial-cli subplebbit start <address>
```

समुदाय अब बिटसोशल नेटवर्क पर लाइव है और किसी भी संगत क्लाइंट से पहुंच योग्य है।
