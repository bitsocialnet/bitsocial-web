# Szablon listy funkcji

Ta strona odzwierciedla szablon JSON dostarczony w pakiecie `docs/agent-playbooks/templates/feature-list.template.json`, dzięki czemu można go przeglądać bezpośrednio w witrynie dokumentów.

```json
{
  "task": "replace-with-task-slug",
  "last_updated": "YYYY-MM-DD",
  "items": [
    {
      "id": "F001",
      "priority": 1,
      "status": "pending",
      "description": "Describe one end-to-end feature or one reviewable task slice.",
      "verification": ["List the command or user-visible check that proves this item works."],
      "files": [],
      "notes": ""
    }
  ]
}
```
