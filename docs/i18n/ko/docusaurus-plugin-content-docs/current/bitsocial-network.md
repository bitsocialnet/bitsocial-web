# Bitsocial Network

Bitsocial Network는 Bitsocial 앱 아래에 위치하게 될 공유 앱체인 계층에 대한 마스터 플랜에서 사용되는 이름입니다.

P2P 소셜 계층을 대체하려는 아이디어는 아닙니다. 이는 소셜 앱이 기존 플랫폼과 대규모로 경쟁할 경우 필요한 공유 경제 및 조정 기본 요소를 제공하는 것입니다.

##

- `.bso`와 같은 분산형 Bitsocial 도메인
- 앱 전반에 걸쳐 공유 유동성
- 내구성 있는 수익 창출 레일
- 금융을 지원한다는 의미입니다. 은행이나 플랫폼이 차단하기 어려운 구조
- 전체 스택을 소유하는 한 회사에 의존하지 않는 네트워크 효과

## 중요한 이유

커뮤니티와 ID를 분산시키는 것이 필요하지만 모든 소셜 미디어를 분산시키는 것만으로는 충분하지 않습니다.

경제 계층이 여전히 소수의 중앙화된 레일에 의존한다면 앱은 여전히 압박을 받거나, 플랫폼을 벗어나거나, 굶어 죽기 쉽습니다. 재정적으로. Bitsocial Network는 종속성의 두 번째 계층에 대해 제안된 답변입니다.

## 나머지 Bitsocial과의 관계

Bitsocial Network는 사용자가 매일 직접 상호 작용하는 제품이 아니라 Bitsocial 앱을 위한 공유 인프라로 제안됩니다.

대중에게 공개되는 결과는 다음과 같습니다.

- 커뮤니티는 P2P 상태를 유지합니다.
- 앱은 유지됩니다. 차별화
- 더 지속 가능한 수익화 및 명명
- 가치는 중앙화된 플랫폼 소유자를 다시 만들지 않고도 생태계를 통해 이동할 수 있습니다

## 이것이 4단계에 있는 이유

마스터 계획은 이를 이전 앱 수준 증명 지점 뒤에 배치합니다:

1. 이미지 보드 분산화
2. Bitsocial 출시 RPC
3. 포럼 분산화
4. 비트소셜 네트워크 출시

실제 앱과 실제 사용자 행동이 이미 네트워크에 존재한다면 생태계 수준 조정 계층이 더 유용하기 때문에 이러한 순서는 의미가 있습니다.
