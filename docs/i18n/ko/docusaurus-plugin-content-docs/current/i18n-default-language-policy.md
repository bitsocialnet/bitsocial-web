# i18n 기본 언어 정책

이제 익명 방문자는 다음 순서로 언어를 해결합니다.

1. `?lang=` 쿼리 매개변수
2. `localStorage`의 저장된 언어 선택기 선택
3. 지역 인식 익명 기본값:
   - 명시적으로 나열된 영어가 많은 지역에서 영어를 강제합니다.
   - 그렇지 않으면 지원되는 로케일 중 하나인 경우 브라우저/장치 언어를 사용하십시오.
4. `en`으로 돌아가기

## 소스 기준선

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) 국가 수준의 영어 능력
- [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) 싱가포르의 경우 5세 이상 거주자의 48.3%가 영어를 가장 자주 사용하는 가정 언어로 사용했습니다.

## 인코딩된 영어-기본 지역

`AT`, `BE`, `CH`, `CZ`, `DE`, `DK`, `FI`, `GR`, `HU`, `MY`, `NL`, `NO`, `PH`, `PL`, `PT`, `RO`, `SE`, `SG`
