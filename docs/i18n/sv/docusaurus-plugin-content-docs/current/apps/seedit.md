# seedit

seedit är ett serverlöst, adminlöst och helt decentraliserat alternativ till gammaldags Reddit, byggt på Bitsocial-protokollet. Det ger användarna fullständigt ägande över sina gemenskaper - vem som helst kan skapa en, och flera gemenskaper kan konkurrera om standardlistningsplatser.

## Nedladdningar

| Plattform  | Länk                                                   |
| ---------- | ------------------------------------------------------ |
| Webb       | [seedit.app](https://seedit.app)                       |
| Webb (ENS) | `seedit.eth` via Brave webbläsare eller IPFS Companion |
| Desktop    | Tillgänglig för Mac, Windows och Linux                 |
| Mobil      | Tillgänglig för Android                                |

## Hur samhällen fungerar

seedit organiserar innehåll i gemenskaper som liknar subreddits. Varje community ägs av dess skapare och fungerar utan centraliserad administration. När flera grupper riktar in sig på samma standardplats kan de tävla om den positionen.

### Skapa en gemenskap

Det finns två sätt att skapa en ny gemenskap:

1. **Skrivbordsklient** — Använd det inbyggda gränssnittet i skrivbordsapplikationen för att konfigurera och konfigurera din community.
2. **bitsocial-cli** — Kör CLI-verktyget för att skapa och hantera gemenskaper från kommandoraden.

### Standardgrupplista

Listan över standardgemenskaper som visas på förstasidan definieras i `default-multisub.json` i listförrådet. För att lägga till din grupp till standardinställningarna, skicka en pull-begäran med din communitys information.
