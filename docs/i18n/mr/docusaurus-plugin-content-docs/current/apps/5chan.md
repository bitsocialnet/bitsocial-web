# 5chan

5chan हा सर्व्हरलेस, ॲडमिनलेस आणि पूर्णपणे विकेंद्रित इमेजबोर्ड आहे जो बिटसोशियल प्रोटोकॉलवर चालतो. विकेंद्रित मालकी सादर करताना हे परिचित इमेजबोर्ड निर्देशिकेच्या संरचनेचे अनुसरण करते — कोणीही एक बोर्ड तयार करू शकतो, आणि एकाधिक बोर्ड मतदान यंत्रणेद्वारे समान निर्देशिका स्लॉटसाठी स्पर्धा करू शकतात.

## डाउनलोड

| प्लॅटफॉर्म | लिंक                               |
| ---------- | ---------------------------------- |
| वेब        | [5chan.app](https://5chan.app)     |
| डेस्कटॉप   | Mac, Windows आणि Linux साठी उपलब्ध |
| मोबाइल     | Android साठी उपलब्ध                |

## बोर्ड कसे कार्य करतात

5chan डिरेक्टरी वापरून क्लासिक बोर्डमध्ये सामग्री व्यवस्थापित करते. `/b/`, `/g/`) पारंपारिक इमेजबोर्डच्या विपरीत जेथे केंद्रीय प्रशासक प्रत्येक बोर्डवर नियंत्रण ठेवतो, 5chan कोणत्याही वापरकर्त्यास त्यांचे स्वतःचे बोर्ड तयार करण्याची आणि पूर्ण मालकीची परवानगी देतो जेव्हा एकाधिक बोर्ड समान निर्देशिका स्लॉटला लक्ष्य करतात, तेव्हा ते मतदानाद्वारे त्या स्थानासाठी स्पर्धा करतात.

### बोर्ड तयार करणे

नवीन बोर्ड तयार करण्यासाठी, तुम्हाला `bitsocial-cli` पीअर-टू-पीअर नोड म्हणून चालवावे लागेल. हे सुनिश्चित करते की तुमचा बोर्ड कोणत्याही केंद्रीय सर्व्हरवर विसंबून न राहता विकेंद्रित पद्धतीने होस्ट केला आहे.

### निर्देशिका असाइनमेंट

निर्देशिका स्लॉट असाइनमेंट (कोणता बोर्ड कोणत्या मार्गावर दिसतो) सध्या `5chan-directories.json` फाइलसाठी GitHub pull requests द्वारे व्यवस्थापित केल्या जातात. ही तात्पुरती प्रक्रिया आहे — भविष्यातील रिलीझमध्ये निर्देशिका असाइनमेंट स्वयंचलितपणे हाताळण्यासाठी in-app board creation आणि pubsub-आधारित मतदानास समर्थन दिले जाईल.

## इंटर्नल्स

हूड अंतर्गत, 5chan त्याच्या प्रोटोकॉल परस्परसंवादासाठी plebbit-js API स्तर वापरते. वरील चेतावणीमध्ये नमूद केल्याप्रमाणे, या अंतर्गत संदर्भांमध्ये अजूनही वारसा नामकरण आहे जे बिटसोशियल रीब्रँडच्या आधीचे आहे.
