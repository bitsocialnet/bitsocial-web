# कमिट आणि इश्यू फॉरमॅट

अर्थपूर्ण कोड बदल सुचवताना किंवा अंमलात आणताना याचा वापर करा.

## कमिट सजेशन फॉरमॅट

- **शीर्षक:** पारंपारिक कमिट शैली, लहान, बॅकटिकमध्ये गुंडाळलेली.
- **शीर्षक:** कार्यप्रदर्शन ऑप्टिमायझेशनसाठी `fix` ऐवजी `perf` वापरा.
- **वर्णन:** समाधानाचे वर्णन करणारी पर्यायी २-३ अनौपचारिक वाक्ये. संक्षिप्त, तांत्रिक, बुलेट पॉइंट नाही.

उदाहरण:

> **कमिट शीर्षक:** `fix: correct date formatting in timezone conversion`
>
> टाइमझोन ऑफसेट योग्यरित्या हाताळण्यासाठी `date-utils.ts` मध्ये `formatDate()` अपडेट केले.

## GitHub इश्यू सजेशन फॉरमॅट

- **शीर्षक:** शक्य तितक्या लहान, बॅकटिकमध्ये गुंडाळलेले.
- **वर्णन:** समस्येचे वर्णन करणारी 2-3 अनौपचारिक वाक्ये (उपाय नाही), जणू काही अद्याप निराकरण झाले नाही.
