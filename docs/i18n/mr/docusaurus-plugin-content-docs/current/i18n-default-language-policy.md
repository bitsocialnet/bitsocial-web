# i18n डीफॉल्ट भाषा धोरण

अनामिक अभ्यागत आता या क्रमाने भाषेचे निराकरण करतात:

1. `?lang=` क्वेरी पॅराम
2. `localStorage`
3. इंग्रजी/प्रदेशात
   - अनॉनिमस-फोर्स-वेअर मधील सेव्ह केलेली भाषा निवडक निवड स्पष्टपणे सूचीबद्ध केलेले इंग्रजी-भारी प्रदेश
   - अन्यथा ब्राउझर/डिव्हाइस भाषा वापरा जेव्हा ती आमच्या समर्थित लोकेलपैकी एक असेल तेव्हा
4. `en`

## स्रोत बेसलाइन

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) देश-स्तरीय इंग्रजी प्रवीणतेसाठी
- [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) सिंगापूरसाठी, जिथे 5+ वयोगटातील 48.3% रहिवाशांसाठी इंग्रजी हे सर्वाधिक बोलले जाणारे घरगुती भाषा होते
