# CLI

:::warning वारसा नामकरण
हे पॅकेज सध्या त्याच्या अपस्ट्रीम अवलंबित्वातून वारशाने मिळालेल्या लेगसी नामकरण पद्धती वापरते. कमांड, आउटपुट आणि कॉन्फिगरेशनमधील "प्लेबिट" चे संदर्भ भविष्यातील प्रकाशनात "बिटसोशियल" वर स्थलांतरित केले जातील. कार्यक्षमता अप्रभावित आहे.
:::

`bitsocial-cli` हे Bitsocial प्रोटोकॉल बॅकएंडशी संवाद साधण्यासाठी कमांड-लाइन साधन आहे. हे तुम्हाला स्थानिक P2P डिमन चालवू देते, समुदाय तयार आणि कॉन्फिगर करू देते आणि सामग्री प्रकाशित करू देते -- सर्व काही टर्मिनलवरून.

हे `plebbit-js` च्या शीर्षस्थानी तयार केले गेले आहे आणि समुदाय निर्मितीसाठी [5chan](/apps/5chan/) आणि [Seedit](/apps/seedit/) आणि node द्वारे वापरले जाते. व्यवस्थापन.

## स्थापना

पूर्व-निर्मित बायनरी Windows, macOS आणि Linux साठी उपलब्ध आहेत. GitHub वरून तुमच्या प्लॅटफॉर्मसाठी नवीनतम रिलीझ डाउनलोड करा:

**[GitHub प्रकाशनांमधून डाउनलोड करा](https://github.com/bitsocialnet/bitsocial-cli/releases)**

डाउनलोड केल्यानंतर, बायनरी एक्झिक्युटेबल बनवा (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## Demon रनिंग

CLI चा सर्वात सामान्य वापर म्हणजे Bitsocial नोड चालवणे. डिमन P2P नेटवर्किंग लेयर सुरू करतो आणि क्लायंट कनेक्ट करू शकणारे स्थानिक API उघड करतो.

```bash
bitsocial-cli daemon
```

पहिल्या लॉन्चवर, डिमॉन **WebUI** ला लिंक करतो, तुमचा नोड, समुदाय आणि सेटिंग्ज व्यवस्थापित करण्यासाठी ब्राउझर-आधारित ग्राफिकल इंटरफेस. टर्मिनल कमांड्सपेक्षा तुम्ही GUI ला प्राधान्य दिल्यास हे उपयुक्त आहे.

## की कमांड

| आज्ञा               | वर्णन                                              |
| ------------------- | -------------------------------------------------- |
| `daemon`            | Bitsocial P2P नोड सुरू करा                         |
| `create subplebbit` | नवीन समुदाय तयार करा                               |
| `subplebbit edit`   | समुदाय सेटिंग्ज अद्यतनित करा (शीर्षक, वर्णन, नियम) |
| `subplebbit list`   | या नोडवर होस्ट केलेले समुदाय सूचीबद्ध करा          |
| `subplebbit start`  | विशिष्ट समुदायाची सेवा सुरू करा                    |
| `subplebbit stop`   | विशिष्ट समुदायाची सेवा देणे थांबवा                 |

उपलब्ध पर्याय आणि ध्वज पाहण्यासाठी `--help` सह कोणतीही आज्ञा चालवा:

```bash
bitsocial-cli daemon --help
bitsocial-cli create subplebbit --help
```

## नमुनेदार कार्यप्रवाह

नवीन समुदाय होस्ट करण्यासाठी एक सामान्य सेटअप प्रवाह:

```bash
# 1. डिमन सुरू करा
bitsocial-cli daemon

# 2. दुसऱ्या टर्मिनलमध्ये, एक समुदाय तयार करा
bitsocial-cli create subplebbit

# 3. समुदाय कॉन्फिगर करा
bitsocial-cli subplebbit edit <address> --title "My Community" --description "A decentralized forum"

# 4. ते सर्व्ह करणे सुरू करा
bitsocial-cli subplebbit start <address>
```

समुदाय आता कोणत्याही बिट वरून संकलित नेटवर्कवर थेट आणि प्रवेशयोग्य आहे क्लायंट.
