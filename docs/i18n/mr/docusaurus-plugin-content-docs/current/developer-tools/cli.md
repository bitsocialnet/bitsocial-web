# CLI

`bitsocial-cli` हे Bitsocial प्रोटोकॉल बॅकएंडशी संवाद साधण्यासाठी कमांड-लाइन साधन आहे. हे तुम्हाला स्थानिक P2P डिमन चालवू देते, समुदाय तयार आणि कॉन्फिगर करू देते आणि सामग्री प्रकाशित करू देते -- सर्व टर्मिनलवरून.

हे सामायिक बिटसोशियल प्रोटोकॉल क्लायंट लेयरच्या वर तयार केले आहे आणि समुदाय निर्मिती आणि नोड व्यवस्थापनासाठी [5chan](/apps/5chan/) आणि [सीडीट](/apps/seedit/) द्वारे वापरले जाते.

## स्थापना

Windows, macOS आणि Linux साठी पूर्व-निर्मित बायनरी उपलब्ध आहेत. GitHub वरून आपल्या प्लॅटफॉर्मसाठी नवीनतम प्रकाशन डाउनलोड करा:

**[GitHub प्रकाशन डाउनलोड करा](https://github.com/bitsocialnet/bitsocial-cli/releases)**

डाउनलोड केल्यानंतर, बायनरी एक्झिक्युटेबल बनवा (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## डिमन चालवणे

CLI चा सर्वात सामान्य वापर म्हणजे Bitsocial नोड चालवणे. डिमन P2P नेटवर्किंग स्तर सुरू करतो आणि क्लायंट कनेक्ट करू शकणारे स्थानिक API उघड करतो.

```bash
bitsocial-cli daemon
```

पहिल्या लाँचवर, डिमन **WebUI** शी लिंक करते, तुमचा नोड, समुदाय आणि सेटिंग्ज व्यवस्थापित करण्यासाठी ब्राउझर-आधारित ग्राफिकल इंटरफेस. टर्मिनल कमांडपेक्षा तुम्ही GUI ला प्राधान्य दिल्यास हे उपयुक्त आहे.

## मुख्य क्रिया

| कृती                        | वर्णन                                              |
| --------------------------- | -------------------------------------------------- |
| डिमन सुरू करा               | Bitsocial P2P नोड लाँच करा                         |
| समुदाय तयार करा             | नवीन समुदाय तयार करा                               |
| समुदाय संपादित करा          | समुदाय सेटिंग्ज अद्यतनित करा (शीर्षक, वर्णन, नियम) |
| स्थानिक समुदायांची यादी करा | या नोडवर होस्ट केलेले समुदाय सूचीबद्ध करा          |
| समुदाय सुरू करा             | विशिष्ट समुदायाची सेवा सुरू करा                    |
| समुदाय थांबवा               | विशिष्ट समुदायाची सेवा करणे थांबवा                 |

तुमच्या इंस्टॉल केलेल्या रिलीझद्वारे उघडकीस आलेल्या वर्तमान कमांडची नावे आणि ध्वज पाहण्यासाठी `--help` सह CLI चालवा:

```bash
bitsocial-cli --help
bitsocial-cli daemon --help
```

## ठराविक कार्यप्रवाह

नवीन समुदाय होस्ट करण्यासाठी एक सामान्य सेटअप प्रवाह:

```bash
# 1. Start the daemon
bitsocial-cli daemon

# 2. In another terminal, inspect the available community-management commands
bitsocial-cli --help
```

तेथून, समुदाय तयार करण्यासाठी, कॉन्फिगर करण्यासाठी आणि सेवा सुरू करण्यासाठी स्थापित रिलीझच्या समुदाय-व्यवस्थापन आदेशांचा वापर करा. एकदा सुरू केल्यानंतर, समुदाय बिटसोशियल नेटवर्कवर थेट असतो आणि सुसंगत क्लायंटकडून प्रवेश करता येतो.
