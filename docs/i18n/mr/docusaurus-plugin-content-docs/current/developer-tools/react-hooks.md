# React Hooks

:::warning वारसा नामकरण
हे पॅकेज सध्या त्याच्या अपस्ट्रीम फोर्कमधून वारशाने मिळालेल्या लेगसी नामकरण पद्धती वापरते. कोड, API आणि कॉन्फिगरेशनमधील "प्लेबिट" चे संदर्भ भविष्यातील प्रकाशनात "बिटसोशियल" वर स्थलांतरित केले जातील. कार्यक्षमतेवर परिणाम होत नाही.
:::

`bitsocial-react-hooks` पॅकेज Bitsocial प्रोटोकॉलशी संवाद साधण्यासाठी परिचित React hooks API प्रदान करते. हे फीड, टिप्पण्या आणि लेखक प्रोफाइल आणणे, खाती व्यवस्थापित करणे, सामग्री प्रकाशित करणे आणि समुदायांची सदस्यता घेणे - सर्व काही केंद्रीय सर्व्हरवर अवलंबून न राहता हाताळते.

ही लायब्ररी [5chan](/apps/5chan/) आणि इतर बिटसोशियल क्लायंट ऍप्लिकेशन्सद्वारे वापरली जाणारी प्राथमिक इंटरफेस आहे.

:::note
`bitsocial-react-hooks` हा `plebbit/plebbit-react-hooks` चा तात्पुरता काटा आहे जो AI-सहाय्यित विकासासाठी राखला जातो. हे npm वर प्रकाशित करण्याऐवजी थेट GitHub वरून वापरले जाते.
:::

## स्थापना

पॅकेज अद्याप npm वर नसल्यामुळे, विशिष्ट कमिट हॅशवर पिन करून ते थेट GitHub वरून स्थापित करा:

```bash
yarn add https://github.com/bitsocialnet/bitsocial-react-hooks.git#<commit-hash>
```

`<commit-hash>` ला तुम्ही लक्ष्य करू इच्छित असलेल्या कमिटसह बदला.

## API विहंगावलोकन

हुक कार्यात्मक श्रेणींमध्ये व्यवस्थापित केले आहेत. खाली प्रत्येक श्रेणीतील सर्वात सामान्यपणे वापरल्या जाणाऱ्या हुकचा सारांश आहे. संपूर्ण स्वाक्षर्या, पॅरामीटर्स आणि रिटर्न प्रकारांसाठी, पहा [GitHub वरील संपूर्ण API संदर्भ](https://github.com/bitsocialnet/bitsocial-react-hooks).

### खाते

स्थानिक वापरकर्ता खाती, ओळख आणि सेटिंग्ज व्यवस्थापित करा.

- `useAccount(accountName?)`) नावाचे खाते (सक्रिय किंवा ऑब्जेक्ट परत करा)
- `useAccounts()` -- स्थानिकरित्या संग्रहित केलेली सर्व खाती परत करते
- `useAccountComments(options?)` -- सक्रिय खात्याद्वारे प्रकाशित टिप्पण्या परत करते

### टिप्पण्या

वैयक्तिक टिप्पण्या आणि थ्रेड मिळवा आणि संवाद साधा.

- `useComment(commentCid?)` -- त्याच्या CID द्वारे एकल टिप्पणी मिळवते
- `useComments(commentCids?)` -- बॅचमध्ये एकाधिक टिप्पण्या मिळवते
- `useEditedComment(comment?)` -- टिप्पणीची नवीनतम संपादित आवृत्ती परत करते

### समुदाय

समुदाय मेटाडेटा आणि सेटिंग्ज पुनर्प्राप्त करा.

- `useSubplebbit(subplebbitAddress?)` -- पत्त्यानुसार समुदाय आणते
- `useSubplebbits(subplebbitAddresses?)` -- एकाधिक समुदाय मिळवते
- `useSubplebbitStats(subplebbitAddress?)` -- सदस्य आणि पोस्ट संख्या परत करतात

### लेखक

लेखक प्रोफाइल आणि मेटाडेटा पहा.

- `useAuthor(authorAddress?)` -- लेखक प्रोफाइल आणते
- `useAuthorComments(options?)` -- विशिष्ट लेखकाच्या टिप्पण्या परत करते
- `useResolvedAuthorAddress(authorAddress?)` -- मानवी-वाचण्यायोग्य पत्त्यावर. त्याचे निराकरण करते. पत्ता

### फीड्स

सामग्री फीड्सची सदस्यता घ्या आणि पृष्ठांकन करा.

- `useFeed(options?)` -- एक किंवा अधिक समुदायांकडील पोस्टचे पृष्ठांकित फीड परत करते
- `useBufferedFeeds(feedOptions?)` -- एकाधिक जलद प्री-बफरसाठी फीड प्रस्तुतीकरण
- `useAuthorFeed(authorAddress?)` -- विशिष्ट लेखकाच्या पोस्टचे फीड परत करते

### क्रिया

सामग्री प्रकाशित करा आणि लेखन ऑपरेशन्स करा.

- `usePublishComment(options?)` -- नवीन टिप्पणी किंवा उत्तर प्रकाशित करा
- `usePublishVote(options?)` -- अपवोट द्या किंवा डाउनवोट
- `useSubscribe(options?)` -- समुदायाची सदस्यता घ्या किंवा सदस्यता रद्द करा

### राज्ये आणि RPC

कनेक्शन स्थितीचे निरीक्षण करा आणि रिमोट बिटसोशियल डिमनशी संवाद साधा.

- `useClientsStates(options?)` -- IPFS/pubsub क्लायंटची कनेक्शन स्थिती परत करते
- `usePlebbitRpcSettings()` -- वर्तमान RPC डिमन कॉन्फिगरेशन परत करते

## डेव्हलपमेंट

स्थानिकरित्या हुक लायब्ररीवर काम करण्यासाठी:

**पूर्वआवश्यकता:** Node.js, Corepack सक्षम, Yarn 4

```bash
git clone https://github.com/bitsocialnet/bitsocial-react-hooks.git
cd bitsocial-react-hooks
corepack enable
yarn install
```

चाचणी आणि बिल्ड कमांडसाठी रेपॉजिटरी README चा संदर्भ घ्या.
