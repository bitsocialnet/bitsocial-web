# Voucher Challenge

व्हाउचर चॅलेंज ही स्पॅम-विरोधी यंत्रणा आहे जी अनन्य व्हाउचर कोडच्या मागे आशयाचे प्रकाशन करते. स्वयंचलित शोधावर विसंबून राहण्याऐवजी, ते समुदाय मालकावर विश्वास हलवते, जो त्यांचा विश्वास असलेल्या लोकांना कोड मॅन्युअली वितरित करतो.

**स्रोत कोड:** [github.com/bitsocialnet/voucher-challenge](https://github.com/bitsocialnet/voucher-challenge)

## हे कसे कार्य करते

1. समुदाय मालक एक किंवा अधिक अद्वितीय व्हाउचर कोड व्युत्पन्न करतो.
2. मालक ते कोड विश्वसनीय लेखकांना त्यांच्या पसंतीच्या चॅनेलद्वारे वितरीत करतो (थेट संदेश, ईमेल, वैयक्तिक इ.).
3. जेव्हा एखादा लेखक प्रकाशित करण्याचा प्रयत्न करतो, तेव्हा आव्हान प्रणाली त्यांना व्हाउचर कोडसाठी सूचित करते.
4. कोड प्रमाणित आहे -- जर तो अस्सल असेल आणि आधीच वापरला गेला नसेल, तर प्रकाशन स्वीकारले जाईल.

प्रत्येक व्हाउचर कोड एका विशिष्ट लेखकाशी जोडलेला असतो एकदा रिडीम केल्यावर, इतरांद्वारे पुन्हा वापरण्यास प्रतिबंध केला जातो.

## ते कधी वापरायचे

व्हाउचर चॅलेंज यासाठी सर्वात योग्य आहे:

- **फक्त-आमंत्रित समुदाय** जेथे सदस्यत्व हेतुपुरस्सर प्रतिबंधित आहे.
- **क्युरेटेड स्पेस** जिथे मालक प्रत्येक सहभागीची वैयक्तिकरित्या तपासणी करतो.
- **उच्च-विश्वास वातावरण** जेथे स्वयंचलित स्पॅम स्कोअरिंग अनावश्यक किंवा अवांछनीय आहे.

कारण त्याला मॅन्युअल कोड वितरण आवश्यक आहे, ते मोठ्या खुल्या समुदायांसाठी स्केल करत नाही. त्या परिस्थितींसाठी, त्याऐवजी [Spam Blocker](./spam-blocker.md) किंवा [EVM Contract Call Challenge](./evm-contract-call.md) विचारात घ्या.

## एकत्रीकरण

व्हाउचर चॅलेंज बिटसोशियल इकोसिस्टममधील इतर अँटी-स्पॅम पॅकेजद्वारे वापरल्या जाणाऱ्या समान आव्हान इंटरफेसमध्ये प्लग इन करते. समुदाय मालक त्यांच्या समुदाय सेटिंग्जद्वारे ते सक्षम करतात आणि लेखक जेव्हा पोस्ट करण्याचा प्रयत्न करतात तेव्हा त्यांना आव्हान स्वयंचलितपणे सादर केले जाते.
