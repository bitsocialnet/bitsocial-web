# Captcha Canvas Challenge

:::warning वारसा नामकरण
हे पॅकेज मूळत: `@plebbit` स्कोप अंतर्गत प्रकाशित केले गेले. त्याचे नाव बदलून `@bitsocial/captcha-canvas-challenge` असे करण्यात आले आहे. जुन्या नावाचे संदर्भ अजूनही जुन्या कागदपत्रांमध्ये किंवा कोडबेसमध्ये दिसू शकतात.
:::

कॅप्चा कॅनव्हास चॅलेंज हे मूळतः `plebbit-js` वरून काढलेले एक स्वतंत्र इमेज कॅप्चा जनरेटर आहे. हे यादृच्छिक मजकूर HTML कॅनव्हासवर रेंडर करते आणि परिणामी प्रतिमा देते, जे समुदाय लेखकांना स्पॅम आव्हान म्हणून सादर करू शकतात.

**स्रोत कोड:** [github.com/bitsocialnet/captcha-canvas-challenge](https://github.com/bitsocialnet/captcha-canvas-challenge)

## आवश्यकता

- **Node.js** >= 22
- **ईएसएम-केवळ** -- हे पॅकेज CommonJS बिल्ड्स पाठवत नाही.
- **रनटाइम पीअर अवलंबित्व:** `@plebbit/plebbit-js` (`@pkc/pkc-js` वर स्थलांतरित करणे)

## स्थापना

```bash
npm install @bitsocial/captcha-canvas-challenge
```

## कॉन्फिगरेशन पर्याय

| पर्याय       | प्रकार   | डीफॉल्ट   | वर्णन                                                         |
| ------------ | -------- | --------- | ------------------------------------------------------------- |
| `characters` | `number` | `6`       | कॅप्चा प्रतिमेमध्ये रेंडर केलेल्या यादृच्छिक वर्णांची संख्या. |
| `height`     | `number` | `100`     | व्युत्पन्न केलेल्या प्रतिमेची उंची पिक्सेलमध्ये.              |
| `width`      | `number` | `300`     | व्युत्पन्न केलेल्या प्रतिमेची रुंदी पिक्सेलमध्ये.             |
| `colors`     | `string` | `#32cf7e` | कॅप्चा मजकूरासाठी वापरलेला प्राथमिक रंग.                      |

## हे कसे कार्य करते

1. जनरेटर कॉन्फिगर केलेल्या लांबीची एक यादृच्छिक स्ट्रिंग निवडतो.
2. ओसीआरला प्रतिकार करण्यासाठी स्ट्रिंग व्हिज्युअल आवाजासह कॅनव्हासवर रेंडर केली जाते.
3. परिणामी प्रतिमा (आणि अपेक्षित कॉलिंग ॲप्लिकेशन नंतर परत येऊ शकते) प्रतिसाद.

पॅकेज शुद्ध प्रतिमा जनरेटर असल्यामुळे, ते स्वतः नेटवर्किंग किंवा सत्र व्यवस्थापन हाताळत नाही. हे एका मोठ्या आव्हान प्रवाहात समाकलित करण्याचा हेतू आहे -- उदाहरणार्थ, [Spam Blocker](./spam-blocker.md) द्वारे समर्थित आव्हान प्रकारांपैकी एक म्हणून.
