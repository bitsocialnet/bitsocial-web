# CLI

:::warning Ονομασία παλαιού τύπου
Αυτό το πακέτο χρησιμοποιεί προς το παρόν συμβάσεις ονομασίας παλαιού τύπου που έχουν κληρονομηθεί από την εξάρτησή του προς τα πάνω. Οι αναφορές στο "plebbit" στις εντολές, την έξοδο και τις ρυθμίσεις παραμέτρων θα μετεγκατασταθούν στο "bitsocial" σε μελλοντική έκδοση. Η λειτουργικότητα δεν επηρεάζεται.
:::

Το `bitsocial-cli` είναι ένα εργαλείο γραμμής εντολών για αλληλεπίδραση με το backend του πρωτοκόλλου Bitsocial. Σας επιτρέπει να εκτελείτε έναν τοπικό δαίμονα P2P, να δημιουργείτε και να διαμορφώνετε κοινότητες και να δημοσιεύετε περιεχόμενο -- όλα από το τερματικό.

Είναι χτισμένο πάνω στο `plebbit-js` και χρησιμοποιείται από [5chan](/apps/5chan/) και [Seedit](/apps/seedit/) για δημιουργία κοινότητας και διαχείριση κόμβων.

## Εγκατάσταση

Τα προκατασκευασμένα δυαδικά αρχεία είναι διαθέσιμα για Windows, macOS και Linux. Κατεβάστε την πιο πρόσφατη έκδοση για την πλατφόρμα σας από το GitHub:

**[Λήψη από Εκδόσεις GitHub](https://github.com/bitsocialnet/bitsocial-cli/releases)**

Μετά τη λήψη, κάντε το δυαδικό εκτελέσιμο (macOS/Linux):

```bash
chmod +x bitsocial-cli
```

## Τρέχοντας τον Δαίμονα

Η πιο κοινή χρήση του CLI είναι η εκτέλεση ενός κόμβου Bitsocial. Ο δαίμονας ξεκινά το επίπεδο δικτύωσης P2P και εκθέτει ένα τοπικό API στο οποίο μπορούν να συνδεθούν οι πελάτες.

```bash
bitsocial-cli daemon
```

Κατά την πρώτη εκκίνηση, ο δαίμονας εξάγει συνδέσμους προς το **WebUI**, μια γραφική διεπαφή που βασίζεται σε πρόγραμμα περιήγησης για τη διαχείριση του κόμβου, των κοινοτήτων και των ρυθμίσεών σας. Αυτό είναι χρήσιμο εάν προτιμάτε ένα GUI από τις εντολές τερματικού.

## Βασικές εντολές

| Εντολή              | Περιγραφή                                                       |
| ------------------- | --------------------------------------------------------------- |
| `daemon`            | Ξεκινήστε τον κόμβο Bitsocial P2P                               |
| `create subplebbit` | Δημιουργία νέας κοινότητας                                      |
| `subplebbit edit`   | Ενημέρωση ρυθμίσεων κοινότητας (τίτλος, περιγραφή, κανόνες)     |
| `subplebbit list`   | Καταχωρίστε τις κοινότητες που φιλοξενούνται σε αυτόν τον κόμβο |
| `subplebbit start`  | Ξεκινήστε να εξυπηρετείτε μια συγκεκριμένη κοινότητα            |
| `subplebbit stop`   | Σταματήστε να εξυπηρετείτε μια συγκεκριμένη κοινότητα           |

Εκτελέστε οποιαδήποτε εντολή με το `--help` για να δείτε διαθέσιμες επιλογές και σημαίες:

```bash
bitsocial-cli daemon --help
bitsocial-cli create subplebbit --help
```

## Τυπική ροή εργασίας

Μια κοινή ροή εγκατάστασης για τη φιλοξενία μιας νέας κοινότητας:

```bash
# 1. Ξεκινήστε τον δαίμονα
bitsocial-cli daemon

# 2. Σε ένα άλλο τερματικό, δημιουργήστε μια κοινότητα
bitsocial-cli create subplebbit

# 3. Διαμορφώστε την κοινότητα
bitsocial-cli subplebbit edit <address> --title "My Community" --description "A decentralized forum"

# 4. Ξεκινήστε να το σερβίρετε
bitsocial-cli subplebbit start <address>
```

Η κοινότητα είναι πλέον ζωντανή στο δίκτυο Bitsocial και είναι προσβάσιμη από οποιονδήποτε συμβατό πελάτη.
