# 5καν

Το 5chan είναι ένας πίνακας εικόνας χωρίς διακομιστή, χωρίς διαχειριστή και πλήρως αποκεντρωμένος που εκτελείται με το πρωτόκολλο Bitsocial. Ακολουθεί τη γνωστή δομή καταλόγου imageboard ενώ εισάγει την αποκεντρωμένη ιδιοκτησία — ο καθένας μπορεί να δημιουργήσει έναν πίνακα και πολλοί πίνακες μπορούν να ανταγωνίζονται για την ίδια υποδοχή καταλόγου μέσω ενός μηχανισμού ψηφοφορίας.

## Λήψεις

| Πλατφόρμα                | Σύνδεσμος                            |
| ------------------------ | ------------------------------------ |
| Web                      | [5chan.app](https://5chan.app)       |
| Επιτραπέζιος υπολογιστής | Διαθέσιμο για Mac, Windows και Linux |
| Κινητό                   | Διαθέσιμο για Android                |

## Πώς λειτουργούν οι σανίδες

Το 5chan οργανώνει το περιεχόμενο σε πίνακες χρησιμοποιώντας μια κλασική διάταξη καταλόγου (π.χ. `/b/`, `/g/`). Σε αντίθεση με τους παραδοσιακούς πίνακες εικόνας όπου ένας κεντρικός διαχειριστής ελέγχει κάθε πίνακα, το 5chan επιτρέπει σε κάθε χρήστη να δημιουργήσει και να κατέχει πλήρως τον δικό του πίνακα. Όταν πολλοί πίνακες στοχεύουν στην ίδια υποδοχή καταλόγου, ανταγωνίζονται για αυτή τη θέση μέσω ψηφοφορίας.

### Δημιουργία πίνακα

Για να δημιουργήσετε έναν νέο πίνακα, πρέπει να εκτελέσετε τον `bitsocial-cli` ως κόμβο peer-to-peer. Αυτό διασφαλίζει ότι η πλακέτα σας φιλοξενείται με αποκεντρωμένο τρόπο χωρίς να βασίζεται σε κανέναν κεντρικό διακομιστή.

### Εργασίες καταλόγου

Επί του παρόντος, η διαχείριση των αναθέσεων θυρίδων καταλόγου (ποιος πίνακας εμφανίζεται σε ποια διαδρομή) γίνεται μέσω αιτημάτων έλξης GitHub στο αρχείο `5chan-directories.json`. Αυτή είναι μια προσωρινή διαδικασία — οι μελλοντικές εκδόσεις θα υποστηρίζουν τη δημιουργία πλακέτας εντός εφαρμογής και την ψηφοφορία που βασίζεται σε pubs για την αυτόματη διαχείριση των αναθέσεων καταλόγου.

## Εσωτερικά

Κάτω από την κουκούλα, το 5chan χρησιμοποιεί το κοινό επίπεδο προγράμματος-πελάτη πρωτοκόλλου Bitsocial για τις αλληλεπιδράσεις του στο δίκτυο. Η εφαρμογή Ιστού στο 5chan.app μπορεί επίσης να εκτελέσει έναν κόμβο Helia στο πρόγραμμα περιήγησης όταν το πρόγραμμα περιήγησης P2P είναι ενεργοποιημένο από τις Ρυθμίσεις για προχωρημένους, ώστε οι αναγνώστες να μπορούν να φορτώνουν από ομοτίμους χωρίς κεντρική πύλη IPFS. Δείτε την ενότητα P2P του προγράμματος περιήγησης στις σημειώσεις πρωτοκόλλου peer-to-peer.
