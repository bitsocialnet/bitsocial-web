# i18n Προεπιλεγμένη πολιτική γλώσσας

Οι ανώνυμοι επισκέπτες επιλύουν τώρα τη γλώσσα με αυτήν τη σειρά:

1. `?lang=` παράμετρος ερωτήματος
2. Αποθηκευμένη επιλογή επιλογέα γλώσσας από `localStorage`
3. Ανώνυμη προεπιλογή με γνώση της περιοχής:
   - επιβάλετε τα αγγλικά σε ρητά καταχωρημένες περιοχές με βαριά αγγλικά
   - Διαφορετικά χρησιμοποιήστε τη γλώσσα του προγράμματος περιήγησης/συσκευής όταν πρόκειται για μία από τις υποστηριζόμενες τοπικές ρυθμίσεις μας
4. Επιστρέψτε στο `en`

## Βασική γραμμή πηγής

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) για επάρκεια αγγλικών σε επίπεδο χώρας
- [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) για τη Σιγκαπούρη, όπου τα αγγλικά ήταν η πιο συχνά ομιλούμενη μητρική γλώσσα για το 48,3% των κατοίκων ηλικίας 5+

## Κωδικοποιημένες αγγλικές-προεπιλεγμένες περιοχές

`AT`, `BE`, `CH`, `CZ`, `DE`, `DK`, `FI`, `GR`, `HU`, `MY`, `NL`, `NO`, `PH`, `PL`, `PT`, `RO`, `SE`, `SG`
