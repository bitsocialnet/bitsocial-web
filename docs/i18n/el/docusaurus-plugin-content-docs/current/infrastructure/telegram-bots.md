# Telegram Bots

Τα ρομπότ Bitsocial Telegram παρακολουθούν τις λίστες κοινοτήτων πελατών στο δίκτυο Bitsocial και προωθούν αυτόματα νέες αναρτήσεις σε κανάλια Telegram. Κάθε προωθημένο μήνυμα περιλαμβάνει ενσωματωμένα κουμπιά που συνδέονται πίσω στην αρχική ανάρτηση στο 5chan και στο Seedit.

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## Διαθέσιμα Bots

| Bot             | Κατάσταση        | Περιγραφή                                                                            |
| --------------- | ---------------- | ------------------------------------------------------------------------------------ |
| **5chan Feed**  | Ενεργός          | Παρακολουθεί όλους τους καταλόγους 5chan και προωθεί νέες δημοσιεύσεις στο Telegram. |
| **Seedit Feed** | Προγραμματισμένη | Θα παρέχει την ίδια λειτουργικότητα για τις κοινότητες Seedit.                       |

## Ρύθμιση

### Προαπαιτούμενα

- Node.js
- Νήματα
- Ένα διακριτικό bot Telegram (δημιουργήστε ένα μέσω [BotFather](https://t.me/BotFather))

### Εγκατάσταση

Κλωνοποιήστε το αποθετήριο και εγκαταστήστε τις εξαρτήσεις:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### Διαμόρφωση

Δημιουργήστε ένα αρχείο `.env` στη ρίζα του έργου με το διακριτικό bot σας:

```env
BOT_TOKEN=your_telegram_bot_token
```

### Τρέξιμο

Ξεκινήστε το bot αφού ρυθμίσετε το περιβάλλον σας:

```bash
yarn start
```

## Μορφή ανάρτησης

Όταν το bot προωθεί μια ανάρτηση στο Telegram, περιλαμβάνει δύο ενσωματωμένα κουμπιά:

- **Προβολή στο 5chan** -- Ανοίγει τη δημοσίευση στο πρόγραμμα-πελάτη ιστού 5chan.
- **Προβολή στο Seedit** -- Ανοίγει την ανάρτηση στο πρόγραμμα-πελάτη ιστού Seedit.

Αυτό επιτρέπει στους συνδρομητές του Telegram να μεταβούν απευθείας στο πλήρες νήμα συζήτησης για όποιον πελάτη προτιμούν.
