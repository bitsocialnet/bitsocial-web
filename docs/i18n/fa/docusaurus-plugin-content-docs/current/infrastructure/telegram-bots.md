# Telegram Bots

ربات‌های تلگرام Bitsocial لیست‌های جامعه مشتریان را در شبکه Bitsocial نظارت می‌کنند و پست‌های جدید را به‌طور خودکار به کانال‌های تلگرام ارسال می‌کنند. هر پیام فوروارد شده شامل دکمه‌های درون خطی است که به پست اصلی در 5chan و Seedit پیوند می‌دهند.

- **GitHub**: [bitsocialnet/bitsocial-telegram-bots](https://github.com/bitsocialnet/bitsocial-telegram-bots)

## ربات های موجود

| ربات           | وضعیت               | توضیحات                                                                             |
| -------------- | ------------------- | ----------------------------------------------------------------------------------- |
| **فید 5chan**  | فعال                | همه دایرکتوری های 5chan را مانیتور می کند و پست های جدید را به تلگرام ارسال می کند. |
| **فید Seedit** | برنامه ریزی شده است | عملکرد مشابهی را برای جوامع Seedit ارائه می دهد.                                    |

## راه اندازی

### پیش نیازها

- Node.js
- نخ
- یک توکن ربات تلگرام (ایجاد یکی از طریق [BotFather](https://t.me/BotFather))

### نصب و راه اندازی

مخزن را کلون کنید و وابستگی ها را نصب کنید:

```bash
git clone https://github.com/bitsocialnet/bitsocial-telegram-bots.git
cd bitsocial-telegram-bots
yarn install
```

### پیکربندی

با توکن ربات خود یک فایل `.env` در ریشه پروژه ایجاد کنید:

```env
BOT_TOKEN=your_telegram_bot_token
```

### در حال دویدن

ربات را پس از پیکربندی محیط خود راه اندازی کنید:

```bash
yarn start
```

## فرمت پست

هنگامی که ربات یک پست را به تلگرام فوروارد می کند، شامل دو دکمه درون خطی می شود:

- **مشاهده در 5chan** -- پست را در سرویس گیرنده وب 5chan باز می کند.
- **مشاهده در Seedit** -- پست را در سرویس گیرنده وب Seedit باز می کند.

این به مشترکین تلگرام این امکان را می دهد که مستقیماً به گفتگوی کامل در مورد هر مشتری که ترجیح می دهند بپردازند.
