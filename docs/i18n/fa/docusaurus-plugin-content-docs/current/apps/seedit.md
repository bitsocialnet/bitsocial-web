# seedit

seedit یک جایگزین بدون سرور، بدون مدیریت و کاملاً غیرمتمرکز برای Reddit به سبک قدیمی است که بر اساس پروتکل Bitsocial ساخته شده است. این به کاربران مالکیت کامل بر جوامع خود را می دهد - هر کسی می تواند یک انجمن ایجاد کند و چندین انجمن می توانند برای اسلات های لیست پیش فرض رقابت کنند.

| موبایل   | موجود برای اندروید                                  |

## چگونه جوامع کار می کنند

sedit محتوا را در جوامعی شبیه به subreddits سازماندهی می کند. هر جامعه ای متعلق به خالق خود است و بدون مدیریت متمرکز عمل می کند. وقتی چندین انجمن یک اسلات پیش‌فرض را هدف قرار می‌دهند، می‌توانند برای آن موقعیت رقابت کنند.

### ایجاد یک جامعه

دو راه برای ایجاد یک جامعه جدید وجود دارد:

1. **مشتری دسکتاپ** — از رابط کاربری گرافیکی داخلی در برنامه دسکتاپ برای راه اندازی و پیکربندی انجمن خود استفاده کنید.
2. **bitsocial-cli** - ابزار CLI را برای ایجاد و مدیریت جوامع از خط فرمان اجرا کنید.

### لیست انجمن پیش فرض

لیست انجمن های پیش فرض که در صفحه اول ظاهر می شود در `default-multisub.json` در مخزن لیست ها تعریف شده است. برای افزودن انجمن خود به پیش‌فرض‌ها، یک درخواست کشش با جزئیات انجمن خود ارسال کنید.
