# خط مشی زبان پیش فرض i18n

بازدیدکنندگان ناشناس اکنون زبان را به ترتیب زیر حل می کنند:

1. `?lang=` پارامتر پرس و جو
2. انتخاب انتخابگر زبان ذخیره شده از `localStorage`
3. پیش‌فرض ناشناس آگاه از منطقه:
   - زبان انگلیسی را در مناطقی که به صراحت فهرست شده اند، تحمیل کنید
   - در غیر این صورت از زبان مرورگر/دستگاه زمانی که یکی از زبان های پشتیبانی شده ما است استفاده کنید
4. به `en` برگردید

## مبنا منبع

- [EF English Proficiency Index 2025](https://www.ef.edu/epi/) برای مهارت انگلیسی در سطح کشور
- [Singapore Census 2020](https://www.singstat.gov.sg/modules/infographics/census-2020) برای سنگاپور، جایی که انگلیسی بیشترین صحبت را در خانه برای 48.3٪ از ساکنان بالای 5 سال داشت.

## مناطق انگلیسی-پیش فرض کدگذاری شده

`AT`, `BE`, `CH`, `CZ`, `DE`, `DK`, `FI`, `GR`, `HU`, `MY`, `NL`, `NO`, `PH`, `PL`, `PT`, `RO`, `SE`, `SG`
